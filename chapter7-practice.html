基础1
复数 z = 3 - 4i 的实部与虚部分别是多少？它的共轭复数是什么？
答案：实部3，虚部-4，共轭复数 z = 3 + 4i
复数 z = a + bi 中，a 为实部，b 为虚部（注意是 b 本身，不是 bi）。
z = 3 - 4i = 3 + (-4)i，所以实部 a = 3，虚部 b = -4。
共轭复数定义为实部相同、虚部互为相反数，即 z = 3 + 4i。
易错：不要把虚部写成 -4i，虚部是不带 i 的系数 -4。
基础2
计算：(1 + 2i) + (3 - 5i)。
答案：4 - 3i
复数加法：实部与实部相加，虚部与虚部相加。
(1 + 2i) + (3 - 5i) = (1 + 3) + (2 - 5)i = 4 + (-3)i = 4 - 3i。
基础3
计算：(4 - 3i) - (2 + 7i)。
答案：2 - 10i
复数减法：实部减实部，虚部减虚部。
(4 - 3i) - (2 + 7i) = (4 - 2) + (-3 - 7)i = 2 + (-10)i = 2 - 10i。
基础4
计算：(1 + 2i)(3 - i)。
答案：5 + 5i
按多项式乘法展开，注意 i² = -1。
(1 + 2i)(3 - i) = 1×3 + 1×(-i) + 2i×3 + 2i×(-i)
= 3 - i + 6i - 2i²
= 3 + 5i - 2(-1) = 3 + 5i + 2 = 5 + 5i。
基础5
计算：(2 + i) / (1 - 2i)。
答案：i
复数除法：分母实数化——分子分母同乘分母的共轭复数。
分母 1 - 2i 的共轭为 1 + 2i。
(2 + i)(1 + 2i)    2 + 4i + i + 2i²    2 + 5i - 2    5i
—————— = ————————— = —————— = —— = i
(1 - 2i)(1 + 2i)    1² - (2i)²       1 + 4     5
注意：(1 - 2i)(1 + 2i) = 1² + 2² = 5，因为 i² = -1，所以 - (2i)² = -4(-1) = 4。
基础6
已知复数 z = (m² - 3m - 4) + (m² - 5m - 6)i（m ∈ R），当 m 为何值时，z 为实数？z 为纯虚数？
答案：m = 6 或 m = -1 时 z 为实数；m = 4 时 z 为纯虚数
（1）z 为实数：虚部为 0。
m² - 5m - 6 = 0 ⇒ (m - 6)(m + 1) = 0 ⇒ m = 6 或 m = -1。
（2）z 为纯虚数：实部为 0 且虚部不为 0。
实部为 0：m² - 3m - 4 = 0 ⇒ (m - 4)(m + 1) = 0 ⇒ m = 4 或 m = -1。
虚部不为 0：m² - 5m - 6 ≠ 0 ⇒ m ≠ 6 且 m ≠ -1。
取交集：m = 4（m = -1 时虚部也为 0，此时 z = 0，既是实数也是纯虚数？通常我们说 0 既不是纯虚数也不是实数？实际上 0 既是实数也是纯虚数（有争议），但按教材定义，纯虚数要求虚部不为 0，所以 m = -1 时 z = 0 不是纯虚数）。
所以 m = 4 时 z 为纯虚数。
基础7
计算 i + i² + i³ + i⁴ + i⁵。
答案：i
利用 i 的幂的周期性：i⁴ⁿ = 1，i⁴ⁿ⁺¹ = i，i⁴ⁿ⁺² = -1，i⁴ⁿ⁺³ = -i。
i¹ = i，i² = -1，i³ = -i，i⁴ = 1，i⁵ = i。
i + i² + i³ + i⁴ + i⁵ = i + (-1) + (-i) + 1 + i = i。
简便方法：i + i² + i³ + i⁴ = 0（每四个一组和为 0），所以原式 = 0 + i⁵ = i。
基础8
在复平面内，复数 z = 2 - 3i 对应的点在哪个象限？
答案：第四象限
复数 z = a + bi 对应复平面内的点 (a, b)。
z = 2 - 3i 对应点 (2, -3)。
横坐标 2 > 0，纵坐标 -3 < 0，所以点在第四象限。
基础9
求复数 z = 3 + 4i 的模 |z|。
答案：5
复数 z = a + bi 的模 |z| = √(a² + b²)。
|z| = √(3² + 4²) = √(9 + 16) = √25 = 5。
几何意义：复平面上点 (3, 4) 到原点的距离。
基础10
已知关于 x 的实系数一元二次方程 x² - 4x + 5 = 0，求其复数根。
答案：x = 2 ± i
判别式 Δ = (-4)² - 4×1×5 = 16 - 20 = -4 < 0，方程有共轭复数根。
求根公式：x = [4 ± √(-4)] / 2 = [4 ± 2i] / 2 = 2 ± i。
所以两根为 x₁ = 2 + i，x₂ = 2 - i（互为共轭复数）。
验算：(2 + i)(2 - i) = 4 - i² = 5 ✅；2 + i + 2 - i = 4 ✅。
拔高1复数运算
已知复数 z 满足 z(1 - i) = 2 + 3i，求复数 z 的共轭复数 z。
答案：z = -1/2 - (5/2)i
由 z(1 - i) = 2 + 3i，得 z = (2 + 3i) / (1 - i)。
分母实数化：z = (2 + 3i)(1 + i) / [(1 - i)(1 + i)]
分子：(2 + 3i)(1 + i) = 2 + 2i + 3i + 3i² = 2 + 5i - 3 = -1 + 5i
分母：(1 - i)(1 + i) = 1² + 1² = 2
所以 z = (-1 + 5i) / 2 = -1/2 + (5/2)i
共轭复数 z = -1/2 - (5/2)i。
拔高2复数几何
在复平面内，复数 z 对应的点 Z 满足 |z + 1 - i| = |z - 3 + i|，求点 Z 的轨迹方程。
答案：2x - y - 2 = 0（一条直线）
设 z = x + yi（x, y ∈ R），代入条件：
|x + yi + 1 - i| = |x + yi - 3 + i|
|(x+1) + (y-1)i| = |(x-3) + (y+1)i|
两边平方（模非负）：(x+1)² + (y-1)² = (x-3)² + (y+1)²
展开：x² + 2x + 1 + y² - 2y + 1 = x² - 6x + 9 + y² + 2y + 1
化简：2x - 2y + 2 = -6x + 2y + 10
8x - 4y - 8 = 0 ⇒ 2x - y - 2 = 0
几何解释：到两个定点 A(-1, 1) 和 B(3, -1) 距离相等的点的轨迹，即线段 AB 的中垂线。
拔高3复数性质
已知复数 z 满足 |z| = 1，求 |z - 2 + 2i| 的最大值和最小值。
答案：最大值 √2 + 1，最小值 |√2 - 1| = √2 - 1
由 |z| = 1 知，z 对应复平面上单位圆上的点。
|z - 2 + 2i| = |z - (2 - 2i)| 表示单位圆上的点到定点 A(2, -2) 的距离。
圆心 O(0, 0) 到 A 的距离 d = √(2² + (-2)²) = √8 = 2√2。
圆上点到圆外定点的最大距离 = d + r = 2√2 + 1。
最小距离 = d - r = 2√2 - 1。
所以 |z - 2 + 2i| 的最大值为 2√2 + 1，最小值为 2√2 - 1。
注意：A(2, -2) 确实是圆外点，因为 d = 2√2 > 1。
拔高4复数方程
解方程：z² = 3 + 4i（z 为复数）。
答案：z = 2 + i 或 z = -2 - i
设 z = a + bi（a, b ∈ R），则 z² = (a + bi)² = a² - b² + 2abi。
由 a² - b² + 2abi = 3 + 4i，得方程组：
⎧ a² - b² = 3    ①
⎨ 2ab = 4        ②
由②得 b = 2/a，代入①：a² - (4/a²) = 3
两边乘 a²：a⁴ - 4 = 3a² ⇒ a⁴ - 3a² - 4 = 0
令 u = a² ≥ 0：u² - 3u - 4 = 0 ⇒ (u - 4)(u + 1) = 0
u = 4（舍去 u = -1），所以 a² = 4 ⇒ a = ±2。
a = 2 时 b = 1；a = -2 时 b = -1。
所以 z = 2 + i 或 z = -2 - i。
验算：(2 + i)² = 4 + 4i + i² = 3 + 4i ✅。
拔高5模与共轭
已知复数 z 满足 z · z + 2iz = 3 + 2i，求复数 z。
答案：z = 1 + i 或 z = -1 + i
设 z = a + bi（a, b ∈ R），则 z = a - bi。
z · z = (a + bi)(a - bi) = a² + b² = |z|²。
代入方程：a² + b² + 2i(a - bi) = 3 + 2i
a² + b² + 2ai - 2bi² = 3 + 2i
a² + b² + 2ai + 2b = 3 + 2i
(a² + b² + 2b) + 2ai = 3 + 2i
由复数相等得：
⎧ a² + b² + 2b = 3    ①
⎨ 2a = 2                ②
由②得 a = 1，代入①：1 + b² + 2b = 3 ⇒ b² + 2b - 2 = 0
b = [-2 ± √(4 + 8)] / 2 = [-2 ± √12] / 2 = [-2 ± 2√3] / 2 = -1 ± √3
所以 z = 1 + (-1 + √3)i 或 z = 1 + (-1 - √3)i。
即 z = 1 + (√3 - 1)i 或 z = 1 - (1 + √3)i。
拔高6实系数方程
已知关于 x 的实系数方程 x² - 2mx + m² + 4 = 0 有复数根，且两根的模之和为 2√5，求实数 m 的值。
答案：m = ±1
判别式 Δ = 4m² - 4(m² + 4) = 4m² - 4m² - 16 = -16 < 0，恒有复数根。
实系数二次方程的复数根互为共轭，设两根为 α 和 α。
由韦达定理：α + α = 2m，α · α = m² + 4。
注意 |α| = |α|，且 |α|² = α · α = m² + 4。
两根的模之和为 2|α| = 2√5 ⇒ |α| = √5。
所以 |α|² = 5 ⇒ m² + 4 = 5 ⇒ m² = 1 ⇒ m = ±1。
验算：m = 1 时方程为 x² - 2x + 5 = 0，根为 1 ± 2i，模均为 √5 ✅。
m = -1 时方程为 x² + 2x + 5 = 0，根为 -1 ± 2i，模均为 √5 ✅。
拔高7复数与几何
在复平面内，复数 z₁, z₂, z₃ 对应的点构成等边三角形，且 z₁ = 1，z₂ = 2 + i，求 z₃。
答案：z₃ = (3 + √3)/2 + (1 - √3)/2 · i 或 z₃ = (3 - √3)/2 + (1 + √3)/2 · i
设 z₃ = x + yi。由等边三角形，边长相等：
|z₂ - z₁| = |z₃ - z₁| = |z₃ - z₂|。
|z₂ - z₁| = |(2 + i) - 1| = |1 + i| = √2。
所以 |z₃ - 1| = √2 且 |z₃ - (2 + i)| = √2。
即 (x - 1)² + y² = 2    ①
且 (x - 2)² + (y - 1)² = 2    ②
② - ①：(x - 2)² - (x - 1)² + (y - 1)² - y² = 0
展开：(x² - 4x + 4) - (x² - 2x + 1) + (y² - 2y + 1) - y² = 0
-4x + 4 + 2x - 1 - 2y + 1 = 0 ⇒ -2x - 2y + 4 = 0 ⇒ x + y = 2
代入①：(x - 1)² + (2 - x)² = 2
x² - 2x + 1 + x² - 4x + 4 = 2 ⇒ 2x² - 6x + 3 = 0
x = [6 ± √(36 - 24)] / 4 = [6 ± √12] / 4 = [6 ± 2√3] / 4 = (3 ± √3) / 2
x = (3 + √3)/2 时 y = 2 - x = (1 - √3)/2
x = (3 - √3)/2 时 y = 2 - x = (1 + √3)/2
所以 z₃ = (3 + √3)/2 + (1 - √3)/2 · i 或 z₃ = (3 - √3)/2 + (1 + √3)/2 · i。
拔高8复数与三角
已知复数 z = cos θ + i sin θ（θ ∈ R），求证：1 + z + z² + … + zⁿ = (1 - zⁿ⁺¹) / (1 - z)（z ≠ 1）。
证明见解析
这是等比数列求和公式在复数域中的直接应用。
设 S = 1 + z + z² + … + zⁿ。
则 zS = z + z² + z³ + … + zⁿ⁺¹。
相减：S - zS = 1 - zⁿ⁺¹
(1 - z)S = 1 - zⁿ⁺¹
因为 z ≠ 1（即 θ ≠ 2kπ），所以 1 - z ≠ 0。
所以 S = (1 - zⁿ⁺¹) / (1 - z)。证毕。
进一步地，当 z = cos θ + i sin θ 时，代入可得：
1 + cos θ + cos 2θ + … + cos nθ = [sin((n+1)θ/2) · cos(nθ/2)] / sin(θ/2)
sin θ + sin 2θ + … + sin nθ = [sin((n+1)θ/2) · sin(nθ/2)] / sin(θ/2)
这是复数在三角恒等式中的重要应用——棣莫弗定理与单位根。
拔高9综合推理
已知复数 z 满足 |z| = 1 且 z² + 2z + 1 = 0，求复数 z。
答案：z = -1 或 z = (-1 ± √3 i)/2
由 |z| = 1，有 z · z = 1，即 z = 1/z。
代入方程：z² + 2(1/z) + 1 = 0
两边乘 z（z ≠ 0）：z³ + 2 + z = 0 ⇒ z³ + z + 2 = 0
因式分解：(z + 1)(z² - z + 2) = 0
① z + 1 = 0 ⇒ z = -1，满足 |z| = 1 ✅。
② z² - z + 2 = 0 ⇒ z = (1 ± √(1 - 8))/2 = (1 ± √7 i)/2
但 |(1 ± √7 i)/2| = √(1 + 7)/2 = √8/2 = √2 ≠ 1 ❌，不满足 |z| = 1，舍去。
等等，让我换一种方法重新做。
由 |z| = 1，设 z = cos θ + i sin θ，则 z = cos θ - i sin θ。
代入 z² + 2z + 1 = 0：
cos 2θ + i sin 2θ + 2(cos θ - i sin θ) + 1 = 0
实部：cos 2θ + 2 cos θ + 1 = 0
虚部：sin 2θ - 2 sin θ = 0
由虚部：2 sin θ cos θ - 2 sin θ = 2 sin θ(cos θ - 1) = 0
所以 sin θ = 0 或 cos θ = 1。
情况1：sin θ = 0 ⇒ θ = kπ，cos θ = ±1
代入实部：cos 2θ + 2 cos θ + 1 = 0
当 cos θ = 1（θ = 2kπ）时，cos 2θ = 1，1 + 2 + 1 = 4 ≠ 0 ❌
当 cos θ = -1（θ = π + 2kπ）时，cos 2θ = 1，1 - 2 + 1 = 0 ✅
所以 z = -1。
情况2：cos θ = 1 ⇒ θ = 2kπ，z = 1
代入：1 + 2 + 1 = 4 ≠ 0 ❌
所以仅有解 z = -1。
拔高10综合应用
已知复数 z₁ = 1 + i，z₂ = a + bi（a, b ∈ R），且满足 z₁ · z₂ 为纯虚数，|z₂| = √10，求 z₂ 的值。
答案：z₂ = √5 + √5 i 或 z₂ = -√5 - √5 i
z₁ · z₂ = (1 + i)(a + bi) = (a - b) + (a + b)i。
因为 z₁ · z₂ 为纯虚数，所以实部为 0，虚部不为 0：
a - b = 0 ⇒ a = b    ①
且 a + b ≠ 0 ⇒ 2a ≠ 0 ⇒ a ≠ 0    ②
又 |z₂| = √(a² + b²) = √10 ⇒ a² + b² = 10    ③
将 ① 代入 ③：a² + a² = 10 ⇒ 2a² = 10 ⇒ a² = 5 ⇒ a = ±√5。
所以 b = a = ±√5。
z₂ = √5 + √5 i 或 z₂ = -√5 - √5 i。
验算：z₁ · z₂ = (1+i)(√5+√5i) = √5(1+i)(1+i) = √5(2i) = 2√5 i，纯虚数 ✅。
|z₂| = √(5+5)=√10 ✅。
