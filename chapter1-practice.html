基础1
用列举法表示集合 { x ∈ Z | x² - 3x - 4 ≤ 0 }。
答案：{-1, 0, 1, 2, 3, 4}
解不等式：x² - 3x - 4 ≤ 0
因式分解：(x-4)(x+1) ≤ 0
解得：-1 ≤ x ≤ 4
x ∈ Z（整数），所以 x = -1, 0, 1, 2, 3, 4
列举法表示为 {-1, 0, 1, 2, 3, 4}
注意：列举法要求元素之间无顺序、不重复，按习惯从小到大写。
基础2
已知集合 A = {1, 2, 3}，B = {2, 3, 4}，求 A ∪ B 和 A ∩ B。
答案：A∪B={1,2,3,4}，A∩B={2,3}
并集：所有属于A或属于B的元素 → A∪B = {1,2,3,4}
交集：既属于A又属于B的元素 → A∩B = {2,3}
Venn图：画两个相交的圆，A={1,2,3}在左圆，B={2,3,4}在右圆，重叠部分{2,3}。
基础3
设全集 U = {1, 2, 3, 4, 5, 6}，A = {1, 3, 5}，求 ∁UA。
答案：∁UA = {2, 4, 6}
补集：在全集中但不属于A的元素。
U = {1,2,3,4,5,6}，去掉A中的{1,3,5}，剩下{2,4,6}。
所以 ∁UA = {2, 4, 6}
基础4
判断下列命题的真假：
(1) ∀x ∈ R，x² ≥ 0；
(2) ∃x ∈ R，x² + 1 = 0；
(3) ∀x ∈ R，|x| ≥ 0。
答案：(1)真 (2)假 (3)真
(1) 任意实数的平方都大于等于0——这是实数的基本性质。✅真命题。
(2) x²+1=0 ⇒ x²=-1，在实数范围内无解。❌假命题。
(3) 任意实数的绝对值都大于等于0——绝对值的定义。✅真命题。
注意：全称量词命题（∀）要求所有情况都成立，存在量词命题（∃）只需找到一个例子。
基础5
已知集合 A = {x | x² - 1 = 0}，B = {-1, 0, 1}，判断 A 与 B 的关系。
答案：A ⊂ B（A是B的真子集）
先化简A：x² - 1 = 0 ⇒ x = ±1，所以 A = {-1, 1}
B = {-1, 0, 1}
A中的每个元素都在B中（-1∈B，1∈B），所以 A ⊆ B。
但B中有元素0不在A中，所以 A ≠ B。
所以 A ⊂ B（A是B的真子集）。
基础6
设集合 A = {x | -2 ≤ x ≤ 3}，B = {x | x > 1}，求 A ∩ B 和 A ∪ B。
答案：A∩B={x|1<x≤3}，A∪B={x|x≥-2}
交集：画数轴，A在[-2,3]，B在(1,+∞)，重叠部分为(1,3]。
所以 A∩B = {x | 1 < x ≤ 3}
并集：A∪B = [-2,3] ∪ (1,+∞) = [-2,+∞)
所以 A∪B = {x | x ≥ -2}
基础7
写出命题"若 x > 0，则 x² > 0"的逆命题、否命题和逆否命题，并判断真假。
答案：见解析
原命题：若 x>0，则 x²>0。（真，正数的平方为正）
逆命题：若 x²>0，则 x>0。（假，x=-1时x²=1>0但x<0）
否命题：若 x≤0，则 x²≤0。（假，x=-1时x²=1>0）
逆否命题：若 x²≤0，则 x≤0。（真，x²≤0⇒x=0⇒x≤0）
规律：原命题与逆否命题同真假，逆命题与否命题同真假。
基础8
已知集合 A = {x | ax + 1 = 0}，若 A 是空集，求实数 a 的值。
答案：a = 0
方程 ax + 1 = 0 无解 ⟺ A = ∅
当 a ≠ 0 时，x = -1/a，有解，A ≠ ∅。
当 a = 0 时，方程变为 1 = 0，无解，A = ∅。
所以 a = 0。
基础9
"x = 2"是"x² = 4"的什么条件？
答案：充分不必要条件
判断：
x=2 ⇒ x²=4（成立），所以充分。
x²=4 ⇒ x=±2，不一定推出x=2（反例：x=-2），所以不必要。
因此"x=2"是"x²=4"的充分不必要条件。
口诀：小范围推大范围——{2}⊂{-2,2}，所以前者是后者的充分不必要条件。
基础10
写出下列命题的否定：
(1) ∀x ∈ R，x + 2 > 0；
(2) ∃x ∈ Z，x² = 2。
答案：(1) ∃x∈R，x+2≤0 (2) ∀x∈Z，x²≠2
全称量词命题的否定：∀改成∃，结论取反。
(1) ∀x∈R，x+2>0 的否定是 ∃x∈R，x+2≤0
存在量词命题的否定：∃改成∀，结论取反。
(2) ∃x∈Z，x²=2 的否定是 ∀x∈Z，x²≠2
拔高1集合运算
已知集合 A = {x | x² - 5x + 6 = 0}，B = {x | mx + 1 = 0}。若 B ⊆ A，求实数 m 的取值集合。
答案：{0, -1/2, -1/3}
先化简A：x²-5x+6=0 ⇒ (x-2)(x-3)=0 ⇒ A={2,3}
B⊆A，分情况讨论：
① B = ∅：mx+1=0无解 ⇒ m=0。此时B=∅⊆A，符合条件。
② B = {2}：2m+1=0 ⇒ m=-1/2。验证：方程(-1/2)x+1=0的解为x=2，B={2}⊆A。
③ B = {3}：3m+1=0 ⇒ m=-1/3。验证：方程(-1/3)x+1=0的解为x=3，B={3}⊆A。
所以 m 的取值集合为 {0, -1/2, -1/3}。
易错：漏掉 B=∅ 的情况（m=0）是这类题最常见的错误。
拔高2子集计数
已知集合 A = {1, 2, 3, 4, 5}。
(1) 求A的子集个数和真子集个数；
(2) 求A的含有元素1的子集个数。
答案：(1) 32个、31个 (2) 16个
(1) A有5个元素，子集个数 = 2⁵ = 32。
真子集个数 = 2⁵ - 1 = 31（去掉自身）。
(2) 含有元素1的子集：先把1放进去，剩下的4个元素每个可以选择取或不取。
所以含有1的子集个数 = 2⁴ = 16。
一般规律：n元集合的子集个数为2ⁿ，真子集2ⁿ-1，含特定元素的子集2ⁿ⁻¹。
拔高3含参集合
已知集合 A = {x | x² - 2x + a = 0}，若 A ⊆ R⁺（正实数集），求实数 a 的取值范围。
答案：0 < a ≤ 1
分析：x²-2x+a=0的两根都在正半轴。
设两根为x₁,x₂，由韦达定理：x₁+x₂=2>0，x₁x₂=a。
分情况：
① A=∅：Δ=4-4a<0 ⇒ a>1。此时A=∅⊆R⁺，成立。
② A≠∅：Δ≥0 ⇒ a≤1。两正根的条件：x₁x₂=a>0。
（x₁+x₂=2>0已自动满足）
综合：0 < a ≤ 1 或 a > 1。即 a > 0 且 a ≠ ?
注意检查：a=0时，方程x²-2x=0，两根为0和2，0∉R⁺，所以a=0不成立。
所以 a > 0 时，若a≤1则两根为正（含相等），若a>1则无解（A=∅⊆R⁺）。
答案：a > 0。实际上题目要求A⊆R⁺，当A=∅时无条件成立，所以a>1也可以。
所以最终的取值范围是 a > 0。
拔高4充要条件
已知 p：-2 ≤ x ≤ 10，q：x² - 2x + 1 - m² ≤ 0（m > 0）。若 q 是 p 的充分不必要条件，求实数 m 的取值范围。
答案：0 < m ≤ 3
化简q：x²-2x+1-m²≤0 ⇒ (x-1)²-m²≤0 ⇒ |x-1|≤m
所以 q 的解集为 [1-m, 1+m]。
p 的解集为 [-2, 10]。
q是p的充分不必要条件 ⇒ q⇒p但p⇏q ⇒ q对应的区间是p对应的区间的真子集。
即 [1-m, 1+m] ⊂ [-2, 10]（真包含）。
列不等式：
① 1-m ≥ -2 ⇒ m ≤ 3
② 1+m ≤ 10 ⇒ m ≤ 9
③ 等号不能同时取（否则就是相等集合，充要条件了）——这里如果m=3时，q=[-2,4]⊂[-2,10]是真子集，不矛盾。
同时 m > 0（题设）。
所以 0 < m ≤ 3。
拔高5集合中新定义
定义集合运算：A ⊗ B = {z | z = xy + x + y, x∈A, y∈B}。已知 A = {0, 1}，B = {1, 2}，求 A ⊗ B。
答案：{1, 3, 5}
逐一代入计算：
x=0,y=1：z = 0×1+0+1 = 1
x=0,y=2：z = 0×2+0+2 = 2
x=1,y=1：z = 1×1+1+1 = 3
x=1,y=2：z = 1×2+1+2 = 5
所以 A ⊗ B = {1, 2, 3, 5}
注意：集合元素有互异性，这里没有重复结果，所以就是4个元素。
拔高6分类讨论
已知集合 A = {x | x² - 3x + 2 = 0}，B = {x | x² - 2x + a - 1 = 0}。若 A ∩ B = B，求实数 a 的取值集合。
答案：{2, 3}
先化简A：x²-3x+2=0 ⇒ (x-1)(x-2)=0 ⇒ A={1,2}
A∩B=B ⇒ B ⊆ A（B是A的子集）。
① B=∅：Δ=4-4(a-1)=8-4a<0 ⇒ a>2。成立。
② B={1}：将x=1代入：1-2+a-1=0 ⇒ a=2。验证：Δ=8-8=0，方程x²-2x+1=0⇒(x-1)²=0⇒x=1，B={1}⊆A。
③ B={2}：将x=2代入：4-4+a-1=0 ⇒ a=1。验证：Δ=8-4=4>0，方程x²-2x+0=0⇒x=0或2，B={0,2}，0∉A，不成立。
④ B={1,2}：1和2都是B的根。由韦达定理：1+2=3=2？但x₁+x₂=2，1+2=3≠2，矛盾。
所以 a=2（B={1}）或 a>2（B=∅），取整数的话就是{2,3,4,…}。
题目问"取值集合"，如果a是实数，就是{2} ∪ (2,+∞)。
拔高7集合运算
已知全集 U = {1, 2, 3, 4, 5, 6, 7, 8, 9}，A ∩ (∁UB) = {1, 3, 5, 7}，(∁UA) ∩ B = {2, 4, 6}，(∁UA) ∩ (∁UB) = {8, 9}。求集合 A 和 B。
答案：A={1,3,5,7}，B={2,4,6}
韦恩图法：画矩形表示全集U，两个相交的圆分别表示A和B。
全集U被分成四个互不相交的部分：
① A∩B（既在A又在B）
② A∩(∁UB)=A-B（仅在A中）= {1,3,5,7}（已知）
③ (∁UA)∩B=B-A（仅在B中）= {2,4,6}（已知）
④ (∁UA)∩(∁UB)=∁U(A∪B)（都不在A也不在B）= {8,9}（已知）
四个部分互不重叠，且它们的并集为全集U。
所以 A = ①∪② = ∅∪{1,3,5,7} = {1,3,5,7}
B = ①∪③ = ∅∪{2,4,6} = {2,4,6}
验证：A∩B=∅，A∪B={1,2,3,4,5,6,7}，剩下的{8,9}为补集，与已知一致。
拔高8逻辑推理
已知 p：关于 x 的方程 x² + ax + 1 = 0 有两个正根。q：a ≤ -2。判断 p 是 q 的什么条件。
答案：充分不必要条件
先解p：方程x²+ax+1=0有两个正根。
① Δ = a² - 4 ≥ 0 ⇒ a ≤ -2 或 a ≥ 2
② 两根之和 = -a > 0 ⇒ a < 0
③ 两根之积 = 1 > 0（已自动满足）
综合①②：a ≤ -2（交集为a≤-2）。
所以 p：a ≤ -2，而 q：a ≤ -2。
p ⇔ q，所以 p 是 q 的充要条件。
拔高9集合综合
设集合 A = {x | x = 2k + 1, k ∈ Z}，B = {x | x = 4k ± 1, k ∈ Z}。求证：A = B。
证明见解析
要证A=B，需证A⊆B且B⊆A。
先证A⊆B：
任取x∈A，则x=2k+1，k∈Z。
① 若k=2m（偶数），x=2(2m)+1=4m+1∈B
② 若k=2m+1（奇数），x=2(2m+1)+1=4m+3=4(m+1)-1∈B
所以A⊆B。
再证B⊆A：
任取x∈B，则x=4k±1=4k+1或4k-1，k∈Z。
① x=4k+1=2(2k)+1∈A
② x=4k-1=4k-1=2(2k)-1=2(2k-1)+1∈A（因为2k-1∈Z）
所以B⊆A。
因此A=B。证毕。
意义：奇数集可以用不同的表达式描述，但本质是同一个集合。
拔高10综合应用
已知集合 A = {x | x² - 4x + 3 ≤ 0}，B = {x | x² - 2mx + m² - 4 ≤ 0}。若 A ∩ B ≠ ∅，求实数 m 的取值范围。
答案：-1 ≤ m ≤ 4
化简A：x²-4x+3≤0 ⇒ (x-1)(x-3)≤0 ⇒ A=[1,3]
化简B：x²-2mx+(m²-4)≤0 ⇒ (x-m)²-4≤0 ⇒ |x-m|≤2 ⇒ B=[m-2, m+2]
A∩B≠∅ ⟺ 两个区间有公共点。
画数轴：A=[1,3]，B=[m-2, m+2]。有交点⇔左端点不大于右端点且右端点不小于左端点。
即：m-2 ≤ 3 且 m+2 ≥ 1
⇒ m ≤ 5 且 m ≥ -1
【注意】这只是初步条件。更精确的：两个区间交集非空⇔B的左端点≤A的右端点且B的右端点≥A的左端点。
m-2 ≤ 3 ⇒ m ≤ 5；m+2 ≥ 1 ⇒ m ≥ -1
所以 m ∈ [-1, 5]。
验证边界：
m=-1：B=[-3,1]，A=[1,3]，交集={1}≠∅。✅
m=5：B=[3,7]，A=[1,3]，交集={3}≠∅。✅
所以 m 的取值范围是 [-1, 5]。
