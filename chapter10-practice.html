基础1
从标有1, 2, 3, 4, 5的5张卡片中随机抽取1张，求抽到奇数的概率。
答案：3/5
样本空间 Ω = {1,2,3,4,5}，共有5个基本事件。
事件A"抽到奇数" = {1,3,5}，包含3个基本事件。
由古典概型公式：P(A) = 3/5。
注意：古典概型要求每个基本事件等可能发生，本题中每张卡片被抽到的概率相同，满足条件。
基础2
同时抛掷两枚质地均匀的硬币，求两枚都正面朝上的概率；至少有一枚正面朝上的概率。
答案：1/4；3/4
样本空间（用H表示正面，T表示反面）：Ω = {HH, HT, TH, TT}，共4个等可能基本事件。
(1) A="两枚都正面"={HH}，P(A)=1/4。
(2) B="至少一枚正面"={HH,HT,TH}，P(B)=3/4。
另解：用对立事件。B的对立事件是"两枚都反面"={TT}，P(B)=1/4，所以P(B)=1−1/4=3/4。
基础3
一个袋中有3个红球和2个白球，从袋中任取2个球。求取出的2个球都是红球的概率。
答案：3/10
从5个球中任取2个，总取法数 n(Ω) = C₅² = 10。
事件A"2个都是红球"：从3个红球中取2个，n(A) = C₃² = 3。
P(A) = 3/10。
易错：注意区分排列与组合！这里取球不考虑顺序，用组合数计算。
基础4
某班级有40名学生，其中男生20人，女生20人。现从全班中随机选取2人，求选出的2人中恰有1男1女的概率。
答案：20/39
总取法数 n(Ω) = C₄₀² = 780。
事件A"恰有1男1女"：选1男（C₂₀¹种）× 选1女（C₂₀¹种）= 20 × 20 = 400。
P(A) = 400/780 = 20/39。
检验：P(2男)=C₂₀²/C₄₀² = 19/78，P(2女)=同样19/78，三者之和为19/78+19/78+20/39 = 19+19+40/78 = 1。✅
基础5
某射手每次射击命中目标的概率为0.8，且每次射击相互独立。求该射手连续射击3次，恰好命中2次的概率。
答案：0.384
这是一个3次独立重复试验（博努利试验），每次命中概率p=0.8，不命中q=0.2。
恰好命中2次的概率：P = C₃² × p² × q¹
= 3 × (0.8)² × 0.2
= 3 × 0.64 × 0.2
= 0.384
公式：n次独立重复试验中恰好发生k次的概率为 P(X=k) = Cₙᵏ pᵏ qⁿ⁻ᵏ。
基础6
从数字1, 2, 3, 4中任取两个不同的数字组成两位数，求这个两位数大于30的概率。
答案：1/2
从4个数字中选2个不同数字排成两位数，总共有 A₄² = 4×3 = 12 种。
事件A"两位数大于30"：十位只能取3或4。
① 十位为3：个位可取1,2,4 → 3种（31,32,34）
② 十位为4：个位可取1,2,3 → 3种（41,42,43）
n(A) = 3+3 = 6。
P(A) = 6/12 = 1/2。
注意：组成两位数时顺序重要，用排列而非组合。
基础7
甲、乙两人各掷一颗质地均匀的骰子，求点数之和为7的概率。
答案：1/6
样本空间：甲有6种点数，乙有6种点数，共6×6=36个等可能结果。
点数之和为7的组合：(1,6),(2,5),(3,4),(4,3),(5,2),(6,1)，共6种。
P = 6/36 = 1/6。
拓展：点数之和为2~12的概率呈对称分布，和为7的概率最大（1/6），和为2或12的概率最小（1/36）。
基础8
已知P(A)=0.5，P(B)=0.3，且A与B互斥，求P(A∪B)和P(A∩B)。
答案：P(A∪B)=0.8，P(A∩B)=0
互斥事件：事件A与B互斥意味着A∩B=∅，即两者不能同时发生。
所以 P(A∩B) = 0。
由互斥事件的加法公式：P(A∪B) = P(A) + P(B) = 0.5 + 0.3 = 0.8。
对比：如果A与B不互斥，则 P(A∪B) = P(A)+P(B)−P(A∩B)。
基础9
已知P(A)=0.6，P(B)=0.7，P(A∩B)=0.4，求P(A∪B)。
答案：0.9
由概率的一般加法公式：
P(A∪B) = P(A) + P(B) − P(A∩B)
= 0.6 + 0.7 − 0.4
= 0.9
验证：P(A∪B)=0.9 ≤ 1，合理。
注意：这里P(A)+P(B)=1.3>1，说明A与B不是互斥的，不能直接用加法公式（忽略了重叠部分被重复计算）。
基础10
一个箱子中有10件产品，其中3件次品。从中不放回地依次取2件，求两件都是正品的概率。
答案：7/15
解法一（组合法）：
总数 n(Ω)=C₁₀²=45，取2件正品 n(A)=C₇²=21。
P(A) = 21/45 = 7/15。
解法二（乘法公式）：
第一次取正品的概率：7/10。
第一次取正品后，剩余9件中正品6件，第二次取正品的概率：6/9 = 2/3。
P = 7/10 × 2/3 = 14/30 = 7/15。
易错："不放回"与"放回"不同。放回时每次独立，概率不变；不放回时条件概率会变化。
拔高1条件概率
某家庭有2个孩子，已知至少有一个是女孩，求另一个也是女孩的概率。（假设生男生女等可能）
答案：1/3
易错警示：很多人会直觉地回答1/2，但正确答案是1/3。
样本空间（用B表示男孩，G表示女孩）：Ω = {BB, BG, GB, GG}，4个等可能结果。
事件A"至少有一个女孩" = {BG, GB, GG}，n(A)=3。
事件B"两个都是女孩" = {GG}。
在A发生的条件下B发生的概率：
P(B|A) = P(A∩B) / P(A) = (1/4) / (3/4) = 1/3。
思路：条件概率缩小了样本空间，从原来的4个结果缩小到3个（排除BB），其中只有GG满足"另一个也是女孩"。
拔高2独立事件
甲、乙两人各进行射击一次。甲命中概率0.9，乙命中概率0.8，且两人是否命中相互独立。求：(1)两人都命中的概率；(2)至少有一人命中的概率；(3)恰好有一人命中的概率。
答案：(1) 0.72 (2) 0.98 (3) 0.26
设A="甲命中"，B="乙命中"，A与B独立。
(1) P(A∩B) = P(A)·P(B) = 0.9×0.8 = 0.72。
(2) 方法一：P(A∪B)=P(A)+P(B)−P(A∩B)=0.9+0.8−0.72=0.98。
方法二（对立事件）：P(至少一人命中) = 1 − P(都未命中) = 1 − (1−0.9)×(1−0.8) = 1 − 0.1×0.2 = 1 − 0.02 = 0.98。
(3) P(恰好一人命中) = P(A∩B) + P(A∩B) = 0.9×0.2 + 0.1×0.8 = 0.18 + 0.08 = 0.26。
验证：0.72(都中) + 0.26(恰一中) + 0.02(都不中) = 1。✅
拔高3古典概型
从0, 1, 2, 3, 4, 5中任取3个不同的数字，组成一个三位数，求这个三位数是偶数的概率。
答案：13/25
总情况数：百位不能为0，所以总数为 5×5×4 = 100个三位数。
事件A"三位数为偶数"：个位必须是偶数（0,2,4），分两种情况：
① 个位为0：百位从1~5中选（5种），十位从剩下4个中选（4种）。共5×4=20个。
② 个位为2或4：
个位：2种选择（2或4）。
百位：不能为0且不能与个位重复，有4种选择（从剩下的非0数中选）。
十位：从剩下的4个数中选（包括0），有4种。
共 2×4×4 = 32个。
n(A) = 20 + 32 = 52。
P(A) = 52/100 = 13/25。
易错：三位数百位不能为0，这是一个关键约束。
拔高4概率基本性质
已知P(A)=0.4，P(B)=0.5，P(A∪B)=0.7。求：(1)P(A∩B)；(2)P(A∩B)。
答案：(1) 0.2 (2) 0.2
(1) 由加法公式：P(A∪B)=P(A)+P(B)−P(A∩B)
0.7 = 0.4 + 0.5 − P(A∩B)
P(A∩B) = 0.4 + 0.5 − 0.7 = 0.2。
(2) 求P(A∩B)即"A发生且B不发生"的概率。
方法一：P(A) = P(A∩B) + P(A∩B)（全集分解）
0.4 = 0.2 + P(A∩B) ⇒ P(A∩B) = 0.2。
方法二：P(A∩B) = P(A) − P(A∩B) = 0.4 − 0.2 = 0.2。
思维：全集分解（全概率公式的雏形）是解决复杂概率问题的重要思想。
拔高5独立事件+串并联
一个电路由三个独立工作的元件串联而成。元件1、2、3正常工作的概率分别为0.9、0.8、0.7。求电路正常工作的概率。如果改成并联，概率又是多少？
答案：串联0.504；并联0.994
串联：必须所有元件都正常工作，电路才通。
P(串联正常) = P(1正常)·P(2正常)·P(3正常) = 0.9×0.8×0.7 = 0.504。
并联：只要至少有一个元件正常工作，电路就通。
用对立事件：P(并联正常) = 1 − P(所有元件都失效)
= 1 − (1−0.9)×(1−0.8)×(1−0.7)
= 1 − 0.1×0.2×0.3
= 1 − 0.006
= 0.994
对比：串联可靠性低于最差元件，并联可靠性高于最优元件。这就是为什么重要系统需要冗余设计（并联）。
拔高6条件概率+乘法公式
盒中有5个红球和3个白球，不放回地依次取3个球，求第三次才取到白球的概率。
答案：5/28
"第三次才取到白球"意味着前两次取的都是红球，第三次取到白球。
用乘法公式（条件概率链式法则）：
第一次取红球的概率：P₁ = 5/8。
在第一次取红球的条件下，剩余7个球中有4个红球，第二次取红球的概率：P₂ = 4/7。
在前两次都取红球的条件下，剩余6个球中有3个白球，第三次取白球的概率：P₃ = 3/6 = 1/2。
P = 5/8 × 4/7 × 1/2 = 20/112 = 5/28。
另解（组合法）：总取法C₈³=56。"前两个红球后一个白球"即前两个从5个红球中取（C₅²=10），第三个必须是固定一个白球？不对，第三个可以是任意一个白球（C₃¹=3），但注意顺序固定（前两红、第三白），所以 n(A) = C₅²×C₃¹ = 10×3=30。P=30/56=15/28？不对——这算的是"恰好前两红第三白"但没考虑顺序已定。实际上组合法计算不放回有序：A₈³=336，前两红第三白=5×4×3=60，P=60/336=5/28。✅
拔高7全概率公式
某工厂有甲、乙、丙三台设备生产同一种产品，产量分别占总产量的40%、35%、25%，次品率分别为2%、3%、4%。从总产品中随机抽取一件，求抽到次品的概率。
答案：0.0285
设事件A="抽到次品"，B₁、B₂、B₃分别表示"产品来自甲、乙、丙"。
{B₁, B₂, B₃}构成了样本空间的一个完备事件组（划分）。
由全概率公式：
P(A) = P(B₁)P(A|B₁) + P(B₂)P(A|B₂) + P(B₃)P(A|B₃)
P(B₁)=0.4，P(A|B₁)=0.02
P(B₂)=0.35，P(A|B₂)=0.03
P(B₃)=0.25，P(A|B₃)=0.04
P(A) = 0.4×0.02 + 0.35×0.03 + 0.25×0.04
= 0.008 + 0.0105 + 0.01
= 0.0285
理解：全概率公式的本质是"加权平均"——以各事件发生的概率为权重，求条件概率的加权平均。
拔高8贝叶斯公式
接上题，若从总产品中随机抽取一件，经检验是次品。求这件次品来自甲设备的概率。
答案：约0.2807
由贝叶斯公式（逆概率公式）：
P(B₁|A) = P(B₁)·P(A|B₁) / P(A)
已知：P(B₁)=0.4，P(A|B₁)=0.02，P(A)=0.0285（上题已求）。
P(B₁|A) = (0.4×0.02) / 0.0285
= 0.008 / 0.0285
≈ 0.2807（即约28.07%）
对比：虽然甲设备产量最高（40%），但它的次品率最低（2%），所以抽到次品来自甲的概率（28.07%）低于其产量占比。
贝叶斯公式的直观理解：先验概率P(B₁)在看到证据（次品）后被"更新"为后验概率P(B₁|A)。
拔高9复杂古典概型
从1, 2, 3, …, 10中随机选取3个不同的数。求：(1)这3个数能构成等差数列的概率；(2)这3个数能构成等比数列的概率。
答案：(1) 2/15 (2) 1/60
总选法：C₁₀³ = 120。
(1) 等差数列：设三数为 a, a+d, a+2d，其中a≥1，a+2d≤10，d为正整数。
d=1：(1,2,3),(2,3,4),…,(8,9,10) → 8组
d=2：(1,3,5),(2,4,6),…,(6,8,10) → 6组
d=3：(1,4,7),(2,5,8),(3,6,9),(4,7,10) → 4组
d=4：(1,5,9),(2,6,10) → 2组
共计 8+6+4+2 = 20组。
P = 20/120 = 1/6？检查：d=1有8组，d=2有6组，d=3有4组，d=4有2组，共20。20/120=1/6。等等，再核实——20/120=1/6≈0.167。不对，我再列一次：
d=1: (1,2,3)~(8,9,10)=8组 ✓
d=2: (1,3,5)~(6,8,10)=6组 ✓
d=3: (1,4,7)~(4,7,10)=4组 ✓
d=4: (1,5,9),(2,6,10)=2组 ✓
共20组，P=20/120=1/6。但注意：还有递减等差的！递增是上面这些，递减其实对应同样的集合（因为无序），所以不需要再算。所以P(等差)=1/6。
(2) 等比数列：设三数为 a, ar, ar²，r为有理数（公比）。
r=2：(1,2,4),(2,4,8) → 2组
r=3：(1,3,9) → 1组
r=1/2：(4,2,1),(8,4,2) → 跟r=2重复的集合，不算
r=？(1,4,16)超出范围
(4,6,9)：4:6=6:9=2:3，公比1.5，(4,6,9) ✅
(9,6,4)是递减，集合相同。
所以不同的等比数列集合有：(1,2,4),(2,4,8),(1,3,9),(4,6,9) — 共4组？
(2,4,8)是等比 ✓，(1,2,4)✓，(1,3,9)✓，(4,6,9)✓。还有(1,4,16)超出。
验证(4,6,9)：4×9=36，6²=36，等比 ✓。
共4组，P=4/120=1/30。
更正：最终答案 P(等差)=1/6，P(等比)=1/30。
拔高10综合应用
甲、乙两人进行乒乓球比赛，采用五局三胜制。已知甲每局胜的概率为0.6，乙每局胜的概率为0.4，各局比赛相互独立。求甲最终获得比赛胜利的概率。
答案：0.68256
甲获胜的情况：甲以3:0、3:1或3:2获胜。
① 3:0（连胜3局）：
P₀ = (0.6)³ = 0.216。
② 3:1（前3局甲胜2局且乙胜1局，第4局甲胜）：
注意：不能在前3局就让甲赢完（否则是3:0），所以前3局甲恰好赢2局。
P₁ = C₃² × (0.6)² × 0.4 × 0.6
= 3 × 0.36 × 0.4 × 0.6
= 3 × 0.36 × 0.24
= 0.2592
③ 3:2（前4局甲胜2局且乙胜2局，第5局甲胜）：
P₂ = C₄² × (0.6)² × (0.4)² × 0.6
= 6 × 0.36 × 0.16 × 0.6
= 6 × 0.36 × 0.096
= 0.20736
总概率：
P = P₀ + P₁ + P₂ = 0.216 + 0.2592 + 0.20736 = 0.68256
易错：很多人在计算3:1和3:2时，忘记"最后一局必须是甲胜"这个条件，误写成C₄³×(0.6)³×0.4或C₅³×(0.6)³×(0.4)²，这样会把甲提前获胜（比如3:0后继续打）的情况也包含进去。
关键：赛点局（最后一局）必须单独写出，前面的局数用组合数计算恰好赢的局数。
