📈 指数函数与对数函数 · 专项训练

人教A版 · 必修第一册 第四章 | 10道基础题 + 10道拔高题

基础1
计算：3√(27) + 20 + 4-1。
答案：25/4
3√(27) = 3，因为 3³ = 27
20 = 1（任何非零数的零次幂等于1）
4-1 = 1/4
所以 原式 = 3 + 1 + 1/4 = 4 + 1/4 = 17/4？等等：3+1=4，4+0.25=4.25=17/4
答案：17/4
基础2
将指数式 34 = 81 改写成对数式。
答案：log₃81 = 4
指数式 ab = N 等价于对数式 logaN = b
底数a=3，指数b=4，真数N=81
所以 log₃81 = 4
注意：底数3写在log右下，真数81写在log后面，结果等于指数4。
基础3
计算：log₂8 + log₅25 - log₃1。
答案：5
log₂8 = 3，因为 2³ = 8
log₅25 = 2，因为 5² = 25
log₃1 = 0，因为 3⁰ = 1
原式 = 3 + 2 - 0 = 5
基础4
求函数 f(x) = 2x 在 x = -1, 0, 2 处的函数值，并判断该函数的单调性。
答案：f(-1)=1/2, f(0)=1, f(2)=4；在R上单调递增
f(-1) = 2-1 = 1/2
f(0) = 20 = 1
f(2) = 22 = 4
指数函数 y = ax（a>0且a≠1）：当 a>1 时单调递增，当 0<a<1 时单调递减。
本题 a=2>1，所以 f(x)=2x 在 R 上单调递增。
基础5
求函数 f(x) = log₂(x - 1) 的定义域。
答案：(1, +∞)
对数函数 logau 要求真数 u > 0
本题真数为 x - 1，所以 x - 1 > 0
解得 x > 1
定义域为 (1, +∞)
易错：忘记真数必须大于0这个条件。
基础6
化简：lg25 + lg4。
答案：2
lg 表示以10为底的对数，即 lg = log₁₀
运用对数加法法则：lgA + lgB = lg(AB)
lg25 + lg4 = lg(25 × 4) = lg100
lg100 = log₁₀100 = 2，因为 10² = 100
基础7
比较大小：23.1 与 22.9；log₃5 与 log₃3.5。
答案：23.1 > 22.9；log₃5 > log₃3.5
指数函数比较：y=2x 在R上单调递增。
因为 3.1 > 2.9，所以 23.1 > 22.9
对数函数比较：y=log₃x 在(0,+∞)上单调递增（底数3>1）。
因为 5 > 3.5，所以 log₃5 > log₃3.5
基础8
方程 2x = 8 的解是多少？并用对数表示其解。
答案：x = 3；x = log₂8
方法一（化为同底）：8 = 2³，所以 2x = 2³ ⇒ x = 3
方法二（对数定义）：x = log₂8 = 3
两种方法都得到 x = 3
注意：指数方程的核心思想是化为同底或取对数。
基础9
已知 log₃x = -2，求 x 的值。
答案：x = 1/9
由对数定义：log₃x = -2 ⇔ 3-2 = x
3-2 = 1/3² = 1/9
所以 x = 1/9
验证：log₃(1/9) = log₃3-2 = -2，正确。
基础10
某放射性物质每经过1年其质量减少为原来的80%。设初始质量为1单位，求经过t年后剩余质量 y 关于 t 的函数解析式。
答案：y = 0.8t
1年后：y = 1 × 0.8 = 0.8
2年后：y = 0.8 × 0.8 = 0.8²
t年后：y = 0.8t
这是一个典型的指数衰减模型 y = at，其中 a = 0.8 ∈ (0,1)。
应用背景：放射性衰变、药物代谢、人口衰减等问题都可用指数模型描述。
拔高1指数运算
化简：(a2/3b-1)-3 ÷ (a-1b1/2)2，结果用分数指数幂表示。
答案：a-4b2
先处理分子的负幂：(a2/3b-1)-3 = a(2/3)×(-3) · b(-1)×(-3) = a-2b3
再处理分母的幂：(a-1b1/2)2 = a-1×2 · b(1/2)×2 = a-2b1
原式 = (a-2b3) / (a-2b) = a-2-(-2) · b3-1 = a0 · b2 = b2
等等纠正：a-2 / a-2 = a0 = 1，b3/b1 = b2
最终结果：b2
关键：分数指数幂的运算遵循 (am)n = amn，以及同底数幂相除指数相减。
拔高2对数运算
计算：log₂3 × log₃4 × log₄5 × log₅2。
答案：1
使用换底公式：logab = ln b / ln a
原式 = (ln3/ln2) × (ln4/ln3) × (ln5/ln4) × (ln2/ln5)
相邻项约分：ln3消掉、ln4消掉、ln5消掉、剩下 ln2/ln2 = 1
所以结果为 1。
规律：loga₁a₂ × loga₂a₃ × ... × logaₙa₁ = 1（循环相消）
拔高3对数方程
解方程：log₂(x + 1) + log₂(x - 1) = 3。
答案：x = 3
首先确定定义域：x+1>0 且 x-1>0 ⇒ x > 1
利用对数加法：log₂(x+1) + log₂(x-1) = log₂[(x+1)(x-1)] = log₂(x²-1)
方程化为：log₂(x²-1) = 3
化为指数式：x² - 1 = 2³ = 8
x² = 9 ⇒ x = ±3
结合定义域 x > 1，舍去 x = -3
所以 x = 3。
易错：解对数方程必须先确定定义域，解完后再检验，防止增根。
拔高4指数方程
解方程：4x - 3·2x+1 + 8 = 0。
答案：x = 1 或 x = 2
令 t = 2x > 0
则 4x = (2²)x = 22x = (2x)² = t²
3·2x+1 = 3·2·2x = 6t
原方程化为：t² - 6t + 8 = 0
因式分解：(t-2)(t-4) = 0 ⇒ t=2 或 t=4
t=2 ⇒ 2x = 2 ⇒ x = 1
t=4 ⇒ 2x = 4 ⇒ x = 2
关键：换元法将指数方程转化为二次方程，注意 t=2x > 0 的隐含条件。
拔高5对数不等式
解不等式：log1/2(x - 1) > log1/2(3 - x)。
答案：1 < x < 2
定义域：x - 1 > 0 且 3 - x > 0 ⇒ 1 < x < 3
底数为 1/2，0 < 1/2 < 1，所以对数函数 y = log1/2u 在 (0,+∞) 上单调递减。
因此不等式 log1/2(x-1) > log1/2(3-x) 等价于：x - 1 < 3 - x
解得：2x < 4 ⇒ x < 2
结合定义域：1 < x < 2
易错：底数在(0,1)之间时，对数函数单调递减，去掉对数后不等号方向要反转！
拔高6复合函数
求函数 f(x) = log2(x² - 2x - 3) 的单调递增区间。
答案：(3, +∞)
首先求定义域：x² - 2x - 3 > 0
因式分解：(x-3)(x+1) > 0 ⇒ x < -1 或 x > 3
令 u = x² - 2x - 3，则 f(x) = log₂u
外层函数 y = log₂u（底数2>1，单调递增）
内层函数 u = x² - 2x - 3，开口向上，对称轴 x = 1
u 的单调区间：(-∞, 1)单调递减，(1, +∞)单调递增
复合函数的单调性遵循"同增异减"：
在定义域 (-∞, -1) 上：u单调递减，外层递增 ⇒ 复合函数递减
在定义域 (3, +∞) 上：u单调递增，外层递增 ⇒ 复合函数递增
所以单调递增区间为 (3, +∞)
拔高7函数零点
判断函数 f(x) = 2x + x - 4 的零点个数，并说明理由。
答案：1个零点（在(1,2)内）
f(x) = 2x + x - 4 在 R 上连续且可导。
f(1) = 2¹ + 1 - 4 = 2 + 1 - 4 = -1 < 0
f(2) = 2² + 2 - 4 = 4 + 2 - 4 = 2 > 0
由零点存在定理，f(x)在(1,2)内至少有一个零点。
f'(x) = 2xln2 + 1 > 0 恒成立，所以 f(x) 在 R 上严格单调递增。
因此 f(x) 恰好有1个零点。
拔高8参数讨论
已知函数 f(x) = log2(x² - ax + 4) 的定义域为 R，求实数 a 的取值范围。
答案：-4 < a < 4
定义域为R，即对任意x∈R，真数 x² - ax + 4 > 0 恒成立。
二次项系数为1>0（开口向上），所以需判别式 Δ < 0。
Δ = a² - 4×1×4 = a² - 16 < 0
a² < 16 ⇒ -4 < a < 4
关键：真数恒正问题转化为二次函数恒正问题，用判别式处理。
拔高9综合应用
某湖泊的污染程度用污染物浓度C(t)表示，初始浓度为C₀。研究表明污染物浓度以每年减少5%的速度自然降解。问经过多少年，污染物浓度降至初始浓度的20%以下？（lg2≈0.3010）
答案：约32年
每年减少5%，即每年浓度为上一年的95% = 0.95
t年后浓度：C(t) = C₀ × 0.95t
要求 C(t) < 0.2C₀，即 0.95t < 0.2
两边取常用对数：t·lg0.95 < lg0.2
lg0.95 = lg(95/100) = lg95 - 2 = lg(5×19) - 2 = lg5 + lg19 - 2
更简便：lg0.95 = lg(19/20) = lg19 - lg20 ≈ 1.2788 - 1.3010 = -0.0222
lg0.2 = lg(1/5) = -lg5 = -(lg10 - lg2) = -(1 - 0.3010) = -0.6990
t > (-0.6990)/(-0.0222) ≈ 31.49
所以需要约32年。
拔高10综合
已知函数 f(x) = ex - e-x。
(1) 判断 f(x) 的奇偶性；
(2) 判断 f(x) 的单调性并证明；
(3) 解不等式 f(x) < 2。
答案：(1)奇函数 (2)R上递增 (3)x < ln(1+√2)
(1) 奇偶性：
f(-x) = e-x - ex = -(ex - e-x) = -f(x)
所以 f(x) 是奇函数。
(2) 单调性：
法一（定义法）：任取 x₁<x₂，
f(x₂)-f(x₁) = (ex₂-e-x₂) - (ex₁-e-x₁)
= (ex₂-ex₁) + (e-x₁-e-x₂)
因为 ex 递增，e-x 递减，所以 ex₂>ex₁ 且 e-x₁>e-x₂
所以 f(x₂)-f(x₁) > 0，f(x)在R上单调递增。
法二（导数法）：f'(x) = ex + e-x > 0，所以f(x)在R上递增。
(3) 解不等式：
f(x) < 2 ⇒ ex - e-x < 2
令 t = ex > 0，则 t - 1/t < 2
两边乘t：t² - 1 < 2t ⇒ t² - 2t - 1 < 0
解二次不等式：t ∈ (1-√2, 1+√2)
因为 t > 0，所以 t ∈ (0, 1+√2)
即 ex < 1+√2 ⇒ x < ln(1+√2)
