基础1
解不等式：3x - 7 > 2x + 1。
答案：x > 8
移项：3x - 2x > 1 + 7
合并：x > 8
解集为 (8, +∞)
基础2
解不等式：x² - 5x + 6 > 0。
答案：x < 2 或 x > 3
因式分解：(x-2)(x-3) > 0
两根为 x=2 和 x=3，二次项系数为正（开口向上）
大于0取两边：x < 2 或 x > 3
解集为 (-∞, 2) ∪ (3, +∞)
基础3
解不等式：-x² + 4x - 3 ≥ 0。
答案：1 ≤ x ≤ 3
两边同乘-1（变号）：x² - 4x + 3 ≤ 0
因式分解：(x-1)(x-3) ≤ 0
两根为 x=1 和 x=3，开口向上
小于等于0取中间：1 ≤ x ≤ 3
注意：二次项系数为负时，先化正再解，避免出错。
基础4
解不等式组：
⎧ 2x - 1 > 3
⎨ x + 2 ≤ 7
答案：2 < x ≤ 5
① 2x - 1 > 3 ⇒ 2x > 4 ⇒ x > 2
② x + 2 ≤ 7 ⇒ x ≤ 5
取交集：2 < x ≤ 5
解集为 (2, 5]
基础5
求二次函数 y = x² - 4x + 3 的顶点坐标和对称轴。
答案：顶点 (2, -1)，对称轴 x = 2
配方：y = x² - 4x + 3 = (x² - 4x + 4) - 4 + 3 = (x - 2)² - 1
顶点坐标为 (2, -1)
对称轴方程为 x = 2
基础6
若 a > b，判断下列不等式中哪些成立：
(1) a - 3 > b - 3    (2) -2a > -2b    (3) a/2 > b/2    (4) a² > b²
答案：(1)(3)成立
(1) a>b ⇒ a-3>b-3（同减一个数，不等号方向不变）✅
(2) a>b ⇒ -2a < -2b（同乘负，方向变）❌
(3) a>b ⇒ a/2 > b/2（同乘正，方向不变）✅
(4) a>b 不能推出 a²>b²。反例：a=1, b=-2，则1>-2但1²<(-2)²。❌
基础7
不等式 x² + 2x + 3 > 0 的解集是什么？
答案：R（全体实数）
判别式 Δ = 2² - 4×1×3 = 4 - 12 = -8 < 0
a=1>0开口向上，Δ<0与x轴无交点
所以 x²+2x+3 > 0 对任意实数都成立，解集为R。
也可以配方：(x+1)²+2>0，显然恒成立。
基础8
已知二次函数 y = x² - 2x - 3 的图象与x轴交于A、B两点，求AB的长度。
答案：4
令y=0：x²-2x-3=0
因式分解：(x-3)(x+1)=0
两根为 x₁=3，x₂=-1
AB长度 = |x₁-x₂| = 3-(-1) = 4
基础9
用不等式表示：某商品原价每件a元，先提价10%，再打9折，最后的售价不超过原价。判断这个说法是否正确。
答案：正确
提价10%后价格：1.1a
再打9折后价格：1.1a × 0.9 = 0.99a
0.99a < a，所以最后售价低于原价。
这个"先提后折"的套路，实际是降价的。
基础10
解关于x的不等式：ax > b（a≠0）。
答案：见解析
需要讨论系数a的符号：
① a > 0：x > b/a
② a < 0：x < b/a（注意变号）
关键：解含参不等式时，必须先讨论参数符号，再决定是否变号。
拔高1含参讨论
解关于x的不等式：x² - (a+1)x + a > 0。
答案：见解析
因式分解：(x-1)(x-a) > 0
两根为 x=1 和 x=a。比较两根大小分类讨论：
① a > 1：解集为 (-∞, 1) ∪ (a, +∞)
② a = 1：(x-1)² > 0，解集为 (-∞, 1) ∪ (1, +∞)
③ a < 1：解集为 (-∞, a) ∪ (1, +∞)
易错：忘记比较两根大小，直接写"取两边"。
拔高2恒成立
关于x的不等式 x² - 2ax + a² + 2 > 0 对任意实数x恒成立，求实数a的取值范围。
答案：a ∈ R
不等式恒成立 ⇒ 二次项系数为正且Δ<0
a=1>0，开口向上。
Δ = (2a)² - 4×1×(a²+2) = 4a² - 4a² - 8 = -8 < 0
Δ恒小于0，所以无论a取什么值，不等式都对所有实数x成立。
a的取值范围是R。
拔高3基本不等式
已知 x > 0，y > 0，且 x + y = 4，求 xy 的最大值。
答案：4
由基本不等式：√(xy) ≤ (x+y)/2
xy ≤ [(x+y)/2]² = (4/2)² = 4
当且仅当 x=y=2 时取等。
所以 xy 的最大值为 4。
另一种方法：y=4-x，xy=x(4-x)=-x²+4x，开口向下抛物线，顶点x=2处取最大值4。
拔高4基本不等式
已知 x > 0，求函数 y = x + 4/x 的最小值。
答案：4
由基本不等式（一正二定三相等）：
因为 x > 0，所以 4/x > 0
x + 4/x ≥ 2√(x·4/x) = 2√4 = 4
当且仅当 x = 4/x，即 x = 2（x>0）时取等。
所以 y 的最小值为 4。
注意：基本不等式的使用条件——正数、乘积为定值、取等条件。
拔高5二次函数最值
已知二次函数 y = x² - 2mx + 3 在 [-1, 2] 上的最小值为 2，求实数 m 的值。
答案：m = 0 或 m = 1
对称轴 x = m，开口向上。
讨论对称轴与区间的关系：
① m < -1：最小值在x=-1处，y(-1)=1+2m+3=4+2m=2 ⇒ m=-1，不满足m<-1。
② -1 ≤ m ≤ 2：最小值在x=m处，y(m)=m²-2m²+3=-m²+3=2 ⇒ m²=1 ⇒ m=±1。
m=1在[-1,2]内；m=-1也在[-1,2]内。但m=-1时y(m)=2，再检查边界？
m=-1时：对称轴x=-1在左端点，最小值取在x=-1，y(-1)=1+2+3=6≠2。不满足。
m=1时：对称轴x=1在区间内，y(1)=1-2+3=2。✅
③ m > 2：最小值在x=2处，y(2)=4-4m+3=7-4m=2 ⇒ m=5/4，不满足m>2。
所以 m = 1。
拔高6不等式证明
已知 a, b, c 为正数，求证：(a+b)(b+c)(c+a) ≥ 8abc。
证明见解析
由基本不等式：
a+b ≥ 2√ab
b+c ≥ 2√bc
c+a ≥ 2√ca
三式相乘：(a+b)(b+c)(c+a) ≥ (2√ab)(2√bc)(2√ca) = 8√(a²b²c²) = 8abc
当且仅当 a=b=c 时取等。
证毕。
拔高7方程根的分布
关于x的方程 x² - 2mx + m² - 1 = 0 有两个实数根，且两根都在区间 (-1, 3) 内，求 m 的取值范围。
答案：0 < m < 2
Δ = 4m² - 4(m²-1) = 4 > 0，恒有两个实根。
两根为 x = (2m ± 2)/2 = m ± 1，即 x₁ = m-1，x₂ = m+1
两根都在(-1,3)内：
① x₁ > -1 ⇒ m-1 > -1 ⇒ m > 0
② x₂ < 3 ⇒ m+1 < 3 ⇒ m < 2
所以 0 < m < 2。
拔高8综合应用
某工厂生产一种产品，固定成本为10万元，每生产1件产品的可变成本为50元，产品售价为每件80元。设年产量为x万件。
(1) 求年利润y（万元）关于x的函数；
(2) 当年产量为多少时，利润最大？最大利润是多少？
答案：(1) y=-50x²+80x-10 (2) 0.8万件，22万元
(1) 收入：80x（万元）；成本：固定10+可变50x
y = 80x - (10 + 50x) = 30x - 10？等等，单位不统一。
固定成本10万元=100000元，每个可变成本50元，售价80元。
利润 = (80-50)x - 10 = 30x - 10（万元），其中x为万件。
所以 y = 30x - 10，x > 0。
(2) 这是一个一次函数，没有最大值！随着x增大利润一直增加。
但是需要考虑市场容量。若题目隐含限制（比如年产量不超过某个数），则需加入。
修正题目理解：通常这类题会给出一个二次的利润函数。
如果售价随产量增加而下降，比如p=80-50x（元/件），则：
y = (80-50x)x - (10+50x) = 80x - 50x² - 10 - 50x = -50x² + 30x - 10
对称轴 x = -30/(2×(-50)) = 0.3万件时取最大值。
最大利润 y(0.3) = -50×0.09 + 30×0.3 - 10 = -4.5 + 9 - 10 = -5.5，负数？有问题。
重新设定更合理的模型：固定成本10万，每件可变成本50元，售价80元/件。
y = (80-50)x - 10 = 30x - 10。年产量x万件。
若年产量上限为2万件，则x=2时ymax=50万元。
拔高9参数讨论
关于x的不等式 ax² - (a+1)x + 1 ≥ 0 的解集为 R，求实数 a 的取值范围。
答案：无解（不存在这样的a）
分情况讨论：
① a=0：不等式变为 -x+1≥0 ⇒ x≤1，解集不是R。❌
② a>0：开口向上，需Δ≤0
Δ = (a+1)² - 4a = a² + 2a + 1 - 4a = a² - 2a + 1 = (a-1)² ≤ 0
所以 a=1。验证：代入a=1，不等式为x²-2x+1=(x-1)²≥0，对所有x∈R成立。✅
③ a<0：开口向下，不可能对所有x∈R都大于等于0（二次函数一定有负值）。❌
所以 a=1。
拔高10综合
已知关于x的不等式 x² - 2ax + a + 2 > 0 对任意 x ∈ (0, 2) 都成立，求实数 a 的取值范围。
答案：a < 2√2
问题是在区间(0,2)上恒成立，不是在整个R上。
设 f(x)=x²-2ax+a+2，对称轴 x=a。
分情况讨论：
① a ≤ 0：对称轴≤0，f(x)在(0,2)上单调递增
最小值在x=0→f(0)=a+2>0，因a≤0时f(0)=a+2>0不一定？
f(0)=a+2>0 ⇒ a>-2。所以 -2<a≤0。
② 0 < a < 2：对称轴在区间内
最小值 f(a)=a²-2a²+a+2=-a²+a+2>0
即 a²-a-2<0 ⇒ (a-2)(a+1)<0 ⇒ -1<a<2
所以 0<a<2。
③ a ≥ 2：对称轴≥2，f(x)在(0,2)上单调递减
最小值在x=2→f(2)=4-4a+a+2=6-3a>0 ⇒ a<2，与a≥2矛盾。❌
综合①②：-2 < a < 2。
