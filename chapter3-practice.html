基础1函数概念
下列各图中，能表示 y 是 x 的函数的是（ ）。
A. 一个圆    B. 一条竖直直线    C. 一条倾斜直线    D. 一条水平直线
答案：C、D
函数的定义：对定义域内每一个 x，都有唯一的 y 与之对应。
A. 圆：一个 x 对应两个 y，不是函数。❌
B. 竖直直线：x 固定，y 不唯一。❌
C. 倾斜直线：每个 x 对应唯一 y。✅
D. 水平直线：每个 x 对应唯一 y。✅
关键：函数是一个 x 只对应一个 y，不能一对多。
基础2定义域
函数 f(x) = √(x-2) + 1/(x-3) 的定义域是什么？
答案：[2, 3) ∪ (3, +∞)
需要同时满足两个条件：
① 根号内 ≥ 0：x - 2 ≥ 0 ⇒ x ≥ 2
② 分母 ≠ 0：x - 3 ≠ 0 ⇒ x ≠ 3
取交集：x ≥ 2 且 x ≠ 3
注意：定义域用区间或集合表示，不要漏掉分母不为零的条件。
基础3值域
函数 f(x) = x² + 2x + 3，x ∈ [-2, 1] 的值域是（ ）。
答案：[2, 6]
配方：f(x) = (x+1)² + 2，对称轴 x = -1，开口向上。
对称轴 x = -1 在区间 [-2, 1] 内。
最小值：f(-1) = (-1+1)² + 2 = 2
比较端点：f(-2) = 4 - 4 + 3 = 3，f(1) = 1 + 2 + 3 = 6
最大值取较大者：6
值域为 [2, 6]。
基础4函数表示法
已知 f(x) = 2x + 1，g(x) = x²，求 f(g(2)) 和 g(f(2))。
答案：f(g(2)) = 9，g(f(2)) = 25
先求 g(2) = 2² = 4，则 f(g(2)) = f(4) = 2×4 + 1 = 9
先求 f(2) = 2×2 + 1 = 5，则 g(f(2)) = g(5) = 5² = 25
易错：f(g(x)) 与 g(f(x)) 一般不同，注意运算顺序。
基础5单调性判断
判断函数 f(x) = -3x + 2 在 R 上的单调性。
答案：在 R 上单调递减
方法一（定义法）：
任取 x₁ < x₂，计算 f(x₁) - f(x₂) = (-3x₁+2) - (-3x₂+2) = -3(x₁ - x₂)
因为 x₁ < x₂，所以 x₁ - x₂ < 0，则 -3(x₁ - x₂) > 0
即 f(x₁) - f(x₂) > 0，所以 f(x₁) > f(x₂)
由 x₁ < x₂ 推出 f(x₁) > f(x₂)，故函数单调递减。
方法二：一次函数斜率 k = -3 < 0，直接得单调递减。
基础6奇偶性
判断下列函数的奇偶性：
(1) f(x) = x³ + x    (2) f(x) = x² + 1    (3) f(x) = x + 1
答案：(1)奇函数 (2)偶函数 (3)非奇非偶
(1) f(-x) = (-x)³ + (-x) = -x³ - x = -(x³+x) = -f(x)，奇函数。✅
(2) f(-x) = (-x)² + 1 = x² + 1 = f(x)，偶函数。✅
(3) f(-x) = -x + 1 ≠ f(x) 且 ≠ -f(x)，非奇非偶。❌
关键：先判断定义域是否关于原点对称，再验证 f(-x) 与 f(x) 的关系。
基础7分段函数
已知 f(x) = { x + 1, x ≥ 0；x², x < 0 }，求 f(-1) + f(2) 的值。
答案：4
f(-1)：因为 -1 < 0，代入第二段：f(-1) = (-1)² = 1
f(2)：因为 2 ≥ 0，代入第一段：f(2) = 2 + 1 = 3
f(-1) + f(2) = 1 + 3 = 4
易错：分段函数要注意自变量所在区间，选择对应的表达式。
基础8幂函数
下列函数中，属于幂函数的是（ ）。
A. y = 2x³    B. y = x^(1/2)    C. y = 3^x    D. y = x² + 1
答案：B
幂函数的定义：形如 y = x^α（α为常数）的函数。
A. y = 2x³ 系数为2，不是幂函数（幂函数系数必须为1）。❌
B. y = x^(1/2) = √x，符合 y = x^α 的形式（α = 1/2）。✅
C. y = 3^x 是指数函数，不是幂函数。❌
D. y = x² + 1 是二次函数，不是幂函数。❌
基础9函数单调性应用
若函数 f(x) = x² - 2ax + 3 在区间 [1, +∞) 上单调递增，求实数 a 的取值范围。
答案：a ≤ 1
f(x) = x² - 2ax + 3，开口向上，对称轴为 x = a。
二次函数在对称轴右侧单调递增，左侧单调递减。
要在 [1, +∞) 上单调递增，只需对称轴 ≤ 区间的左端点。
即 a ≤ 1。
可以画图帮助理解：对称轴在区间左边，区间整体在对称轴右侧。
基础10奇偶性应用
已知函数 y = f(x) 是偶函数，且 f(3) = 5，f(-2) = 4，求 f(-3) + f(2) 的值。
答案：9
偶函数的性质：f(-x) = f(x)
f(-3) = f(3) = 5
f(2) = f(-2) = 4
f(-3) + f(2) = 5 + 4 = 9
关键：偶函数图象关于y轴对称，奇函数图象关于原点对称，记清性质。
拔高1抽象函数
已知函数 f(x) 满足 f(x+1) = x² - 2x + 3，求 f(x) 的解析式。
答案：f(x) = x² - 4x + 6
方法一（换元法）：令 t = x + 1，则 x = t - 1
f(t) = (t-1)² - 2(t-1) + 3 = t² - 2t + 1 - 2t + 2 + 3 = t² - 4t + 6
所以 f(x) = x² - 4x + 6
方法二（配凑法）：
x² - 2x + 3 = (x+1)² - 4(x+1) + 6
令 x+1 替换为 x，得 f(x) = x² - 4x + 6
易错：换元后注意回代，不要混淆 f(x) 和 f(x+1) 的对应关系。
拔高2复合函数定义域
已知 f(x) 的定义域为 [0, 3]，求函数 g(x) = f(x+1) + f(2x-1) 的定义域。
答案：[1/2, 2]
f(x) 的定义域为 [0, 3]，即 f 括号内的表达式必须在 [0, 3] 内。
① 对于 f(x+1)：0 ≤ x+1 ≤ 3 ⇒ -1 ≤ x ≤ 2
② 对于 f(2x-1)：0 ≤ 2x-1 ≤ 3 ⇒ 1/2 ≤ x ≤ 2
取交集：[-1, 2] ∩ [1/2, 2] = [1/2, 2]
关键：复合函数定义域问题——内层函数的值域要满足外层函数的定义域。
拔高3单调性证明
用定义证明：函数 f(x) = x + 1/x 在 (0, 1) 上单调递减。
证明见解析
任取 0 < x₁ < x₂ < 1，考察 f(x₁) - f(x₂)：
f(x₁) - f(x₂) = (x₁ + 1/x₁) - (x₂ + 1/x₂)
= (x₁ - x₂) + (1/x₁ - 1/x₂)
= (x₁ - x₂) + (x₂ - x₁)/(x₁x₂)
= (x₁ - x₂) · [1 - 1/(x₁x₂)]
因为 x₁ < x₂，所以 x₁ - x₂ < 0
因为 0 < x₁ < x₂ < 1，所以 0 < x₁x₂ < 1，则 1/(x₁x₂) > 1，故 1 - 1/(x₁x₂) < 0
负 × 负 = 正，所以 f(x₁) - f(x₂) > 0，即 f(x₁) > f(x₂)
由 x₁ < x₂ 推出 f(x₁) > f(x₂)，故 f(x) 在 (0, 1) 上单调递减。证毕。
注意：差式变形时，要善于提取公因式 (x₁ - x₂)。
拔高4奇偶性与单调性综合
已知定义在 R 上的奇函数 f(x) 在 [0, +∞) 上单调递增，且 f(2) = 0，求解不等式 f(x) < 0。
答案：(-∞, -2) ∪ (0, 2)
奇函数性质：f(-x) = -f(x)，图象关于原点对称。
已知 f(2) = 0，由奇函数得 f(-2) = -f(2) = 0。
在 [0, +∞) 上单调递增，所以：
当 0 ≤ x < 2 时，f(x) < f(2) = 0
当 x > 2 时，f(x) > f(2) = 0
由对称性，在 (-∞, 0] 上：
当 x < -2 时，f(x) < 0（因为 f(-2)=0 且递增）
当 -2 < x ≤ 0 时，f(x) > 0
整理得 f(x) < 0 的解集为 (-∞, -2) ∪ (0, 2)。
方法：画示意图帮助分析，利用对称性和单调性确定各区间符号。
拔高5函数最值
求函数 f(x) = x + √(1 - x) 的最大值。
答案：5/4
定义域：1 - x ≥ 0 ⇒ x ≤ 1，即 (-∞, 1]。
方法一（换元法）：令 t = √(1 - x)，则 t ≥ 0，且 x = 1 - t²。
代入得 f = (1 - t²) + t = -t² + t + 1
这是关于 t 的二次函数（t ≥ 0），开口向下。
对称轴 t = -1 / (2×(-1)) = 1/2，在 [0, +∞) 内。
最大值在 t = 1/2 处：f_max = -(1/2)² + 1/2 + 1 = -1/4 + 1/2 + 1 = 5/4
方法二（导数法，略）。
关键：含根号的函数常用换元法转化为二次函数求最值，注意新变量的范围。
拔高6幂函数比较大小
比较下列各组数的大小：
(1) 0.2^(0.3) 与 0.3^(0.3)    (2) 0.2^(0.3) 与 0.2^(0.4)
答案：(1) 0.2^(0.3) < 0.3^(0.3) (2) 0.2^(0.3) > 0.2^(0.4)
(1) 比较 0.2^(0.3) 与 0.3^(0.3)：
考察幂函数 y = x^(0.3)，指数 α = 0.3 > 0，在 (0, +∞) 上单调递增。
因为 0.2 < 0.3，所以 0.2^(0.3) < 0.3^(0.3)。
(2) 比较 0.2^(0.3) 与 0.2^(0.4)：
考察指数函数 y = 0.2^x，底数 0 < 0.2 < 1，在 R 上单调递减。
因为 0.3 < 0.4，所以 0.2^(0.3) > 0.2^(0.4)。
关键：比较幂值——指数相同用幂函数单调性，底数相同用指数函数单调性。
拔高7恒成立问题
已知函数 f(x) = x² + ax + 3，对任意 x ∈ [-1, 2] 都有 f(x) ≥ 2 恒成立，求实数 a 的取值范围。
答案：-4 ≤ a ≤ 2
问题等价于 f(x) 在 [-1, 2] 上的最小值 ≥ 2。
f(x) = x² + ax + 3，开口向上，对称轴 x = -a/2。
分三种情况讨论：
① 对称轴 ≤ -1：即 -a/2 ≤ -1 ⇒ a ≥ 2
最小值在 x = -1 处：f(-1) = 1 - a + 3 = 4 - a ≥ 2 ⇒ a ≤ 2
结合 a ≥ 2，得 a = 2。
② 对称轴在 [-1, 2] 内：即 -1 < -a/2 < 2 ⇒ -4 < a < 2
最小值在 x = -a/2 处：f(-a/2) = a²/4 - a²/2 + 3 = -a²/4 + 3 ≥ 2 ⇒ a² ≤ 4 ⇒ -2 ≤ a ≤ 2
结合 -4 < a < 2，得 -2 ≤ a < 2。
③ 对称轴 ≥ 2：即 -a/2 ≥ 2 ⇒ a ≤ -4
最小值在 x = 2 处：f(2) = 4 + 2a + 3 = 7 + 2a ≥ 2 ⇒ a ≥ -5/2
结合 a ≤ -4，无解。
综合①②③：a = 2 或 -2 ≤ a < 2，即 -2 ≤ a ≤ 2。
等等，重新检查：对称轴恰在端点时归入相应区间。
最终结果：-2 ≤ a ≤ 2。
拔高8抽象函数性质
定义在 R 上的函数 f(x) 满足：对任意实数 x, y，都有 f(x+y) = f(x) + f(y)，且 f(1) = 2。
(1) 求 f(0)；
(2) 证明 f(x) 为奇函数；
(3) 求 f(3)。
答案：(1) f(0)=0 (2) 见解析 (3) f(3)=6
(1) 求 f(0)：
令 x = y = 0，得 f(0+0) = f(0) + f(0) ⇒ f(0) = 2f(0) ⇒ f(0) = 0
(2) 证明奇函数：
令 y = -x，得 f(x + (-x)) = f(x) + f(-x)
即 f(0) = f(x) + f(-x) ⇒ 0 = f(x) + f(-x) ⇒ f(-x) = -f(x)
所以 f(x) 为奇函数。证毕。
(3) 求 f(3)：
f(2) = f(1+1) = f(1) + f(1) = 2 + 2 = 4
f(3) = f(2+1) = f(2) + f(1) = 4 + 2 = 6
关键：抽象函数问题常通过赋值法（令 x, y 取特殊值）推导性质。
拔高9函数图象应用
已知函数 f(x) 是定义在 R 上的偶函数，且在 [0, +∞) 上单调递增，f(1) = 0。
解不等式 f(2x - 1) > 0。
答案：x < 0 或 x > 1
f(x) 是偶函数，图象关于 y 轴对称。
在 [0, +∞) 上递增，则在 (-∞, 0] 上递减。
f(1) = 0，由偶函数得 f(-1) = f(1) = 0。
f(2x-1) > 0 相当于 |2x-1| > 1（因为偶函数+递增时，f(u) > f(v) ⇔ |u| > |v|）。
即 |2x - 1| > 1。
解得：2x - 1 < -1 或 2x - 1 > 1
⇒ x < 0 或 x > 1。
技巧：偶函数+单调性可转化为绝对值不等式，大幅简化计算。
拔高10综合应用
已知函数 f(x) = ax³ + bx + 5（a, b 为常数），且 f(3) = 10。
(1) 求 f(-3)；
(2) 若 f(x) 是奇函数，求 a, b 的值。
答案：(1) f(-3) = 0 (2) a = 0, b = 0
(1) 利用奇函数的构造法：
设 g(x) = f(x) - 5 = ax³ + bx，则 g(x) 是奇函数（各项次数均为奇数）。
g(-3) = -g(3)
f(-3) - 5 = -(f(3) - 5) = -(10 - 5) = -5
所以 f(-3) = 0
(2) 若 f(x) 是奇函数：
奇函数要求 f(-x) = -f(x) 对所有 x 成立。
f(-x) = a(-x)³ + b(-x) + 5 = -ax³ - bx + 5
-f(x) = -ax³ - bx - 5
由 f(-x) = -f(x) 得：-ax³ - bx + 5 = -ax³ - bx - 5
⇒ 5 = -5，矛盾。所以不存在这样的 a, b 使 f(x) 为奇函数？
等等，检查：f(x) = ax³ + bx + 5 含有常数项 5。
奇函数必须满足 f(0) = 0，但 f(0) = 5 ≠ 0，所以不可能为奇函数。
因此无解。
纠正理解：若要求 f(x) 是奇函数，则常数项必须为 0，且只有奇次项。
所以 a = 0 且 b = 0 时 f(x) = 5，但 f(x) = 5 是偶函数不是奇函数！
综上，不存在这样的 a, b 使 f(x) 为奇函数。因为奇函数定义域含0时必须过原点(0,0)，而f(0)=5≠0。
最终答案：(1) f(-3)=0 (2) 不存在这样的 a, b。
