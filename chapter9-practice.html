基础1
为了了解某校高一年级500名学生的体重情况，从中抽取50名学生进行测量。下列说法中正确的是（ ）。
A. 500名学生是总体    B. 每个学生是个体
C. 50名学生的体重是一个样本    D. 样本容量是50
答案：D
分析：总体是研究对象的全体——500名学生的体重，不是学生本身。A错。
个体是每个学生的体重，不是每个学生。B错。
样本是抽取的50名学生的体重，C的说法"50名学生"不够准确。
样本容量是样本中包含的个体数目（无单位），即50。D正确。
易错点：总体、个体、样本都是指数据，不是指人。
基础2
下列抽样方法中，属于简单随机抽样的是（ ）。
A. 从全班50人中依次抽取学号为1,2,3,…,10的学生
B. 将全班学生编号，用随机数表法抽取10人
C. 按学号奇偶性抽取
D. 按成绩分层抽取
答案：B
简单随机抽样的特点：每个个体被抽到的机会均等，且逐个抽取、不放回。
A是系统抽样（等距抽样），不是简单随机抽样。
B使用随机数表法，属于简单随机抽样。✅
C按奇偶性，不是等概率（奇偶人数可能不等）。
D是分层抽样。
基础3
某校有高一学生400人、高二学生300人、高三学生300人，现用分层抽样法从全校学生中抽取一个容量为50的样本。高一、高二、高三应各抽取多少人？
答案：高一20人，高二15人，高三15人
全校总人数：400 + 300 + 300 = 1000人。
抽样比例：k = 50 / 1000 = 1/20。
高一：400 × 1/20 = 20人
高二：300 × 1/20 = 15人
高三：300 × 1/20 = 15人
检查：20 + 15 + 15 = 50 ✓
注意：分层抽样中，各层抽取人数按比例计算：
某层抽取数 = 该层人数 × (样本容量 / 总体容量)
基础4
一个样本的数据如下：
12, 15, 18, 20, 22, 25, 28, 30。
求该样本的第25百分位数（即下四分位数）。
答案：15.75
数据已从小到大排列，共 n = 8 个数据。
计算位置：i = n × p% = 8 × 25% = 2。
i = 2 是整数，则第25百分位数为第2个和第3个数据的平均值。
第2个数据是15，第3个数据是18。
第25百分位数 = (15 + 18) / 2 = 16.5
注意：百分位数的计算方法有多种（如插值法），教材常用位置法。
另一种常见算法：位置 i = n × p% = 2，取第 i 个和第 i+1 个的平均值。
如果按教材更常见的公式：i = (n+1) × p% = 9 × 0.25 = 2.25，
则第25百分位数 = 第2个数 + 0.25 × (第3个数 - 第2个数) = 15 + 0.25×3 = 15.75。
基础5
某组数据的方差计算公式为：
s² = [(2-5)² + (4-5)² + (5-5)² + (7-5)² + (7-5)²] / 5
求这组数据的平均数和标准差。
答案：平均数 = 5，标准差 = 2
从公式看出平均数为 5（每个偏差²的中间项）。
样本数据：2, 4, 5, 7, 7。验证平均数 = (2+4+5+7+7)/5 = 25/5 = 5。✅
方差 s² = [(2-5)² + (4-5)² + (5-5)² + (7-5)² + (7-5)²] / 5
= [9 + 1 + 0 + 4 + 4] / 5 = 18 / 5 = 3.6
标准差 s = √s² = √3.6 ≈ 1.897
（注意：分母是n还是n-1，如果是总体方差用n，样本方差用n-1。此处分母为5，是按总体方差计算。）
基础6
为了解某小区居民月用电量情况，随机抽查了50户居民的月用电量，数据如下（单位：度）：
最低40，最高180。若将数据分为7组，则组距大约是多少？
答案：20
极差 = 最大值 - 最小值 = 180 - 40 = 140。
组距 = 极差 / 组数 = 140 / 7 = 20。
取整后组距为20。
分组建议：[40,60), [60,80), [80,100), [100,120), [120,140), [140,160), [160,180]。
实际画频率分布直方图时，组距常取整数以便于操作。
基础7
在频率分布直方图中，小长方形的面积表示什么？所有小长方形面积之和等于多少？
答案：频率；1
在频率分布直方图中：
小长方形面积 = 组距 × (频率/组距) = 频率
纵轴是"频率/组距"，即频率密度。
所有组的频率之和等于1，所以所有小长方形面积之和 = 1。
注意：纵轴不是频率，而是频率/组距！这是最常见的易错点。
基础8
甲、乙两名射击运动员的10次射击成绩（环数）如下：
甲：8, 9, 9, 10, 8, 9, 10, 7, 9, 9
乙：6, 10, 10, 8, 9, 10, 7, 10, 8, 10
谁的成绩更稳定？
答案：甲更稳定
思路：比较方差或标准差，方差越小越稳定。
甲的平均数：(8+9+9+10+8+9+10+7+9+9)/10 = 88/10 = 8.8
甲偏离平方和：0.64+0.04+0.04+1.44+0.64+0.04+1.44+3.24+0.04+0.04 = 7.6
甲方差（样本）：s²(甲) = 7.6/9 ≈ 0.84
乙的平均数：(6+10+10+8+9+10+7+10+8+10)/10 = 88/10 = 8.8
乙偏离平方和：7.84+1.44+1.44+0.64+0.04+1.44+3.24+1.44+0.64+1.44 = 19.6
乙方差（样本）：s²(乙) = 19.6/9 ≈ 2.18
因为 s²(甲) < s²(乙)，所以甲的成绩更稳定。

拔高1分层抽样
某地区有甲、乙、丙三个工厂，分别有职工1000人、1500人、2500人。现用分层抽样法抽取一个容量为200的样本调查工资水平。已知甲厂抽取了40人，但实际甲厂职工人数为800人（数据有误）。请问：
(1) 按原计划的1000人计算，甲厂应抽多少人？
(2) 按实际800人重新计算，甲厂应抽多少人？
(3) 两次抽样结果差异说明了什么？
答案：(1) 40人   (2) 约33人   (3) 各层人数准确性对分层抽样很重要
(1) 原总人数 = 1000+1500+2500 = 5000人
抽样比例 k = 200/5000 = 1/25
甲厂应抽：1000 × 1/25 = 40人 ✅
(2) 实际总人数 = 800+1500+2500 = 4800人
抽样比例 k = 200/4800 = 1/24
甲厂应抽：800 × 1/24 = 100/3 ≈ 33.3，即33人。
(3) 两次结果相差7人，说明如果各层人数信息不准确，
分层抽样的样本结构与总体结构就会产生偏差，
影响样本的代表性。分层抽样的前提是各层人数准确。
拔高2频率分布直方图
某校100名学生的数学成绩的频率分布直方图如下（各组数据已简化）：
分组 [50,60) 频率0.05，[60,70) 频率0.15，[70,80) 频率0.30，
[80,90) 频率0.35，[90,100] 频率0.15。
(1) 求成绩在[70,90)的人数；
(2) 若规定85分及以上为优秀，估计优秀人数。
答案：(1) 65人   (2) 约35人
(1) [70,80)频率0.30，[80,90)频率0.35
[70,90)的频率 = 0.30 + 0.35 = 0.65
人数 = 100 × 0.65 = 65人。
(2) 85分在区间[80,90)内，该组频率0.35，组距10。
区间[80,90)且≥85的部分占该组的比例：(90-85)/10 = 0.5
所以≥85分的频率 ≈ 0.35 × 0.5 = 0.175
再加上[90,100]的频率0.15
优秀频率 = 0.175 + 0.15 = 0.325
估计优秀人数 = 100 × 0.325 ≈ 33人。
注意：利用频率分布直方图估计时，需要假设数据在组内均匀分布。
拔高3方差性质
已知一组数据 x₁, x₂, …, xₙ 的平均数为 \bar{x}，方差为 s²。
若对每个数据作变换 yᵢ = axᵢ + b（a, b为常数，a ≠ 0），
求新数据 y₁, y₂, …, yₙ 的平均数和方差。
答案：平均数 = a\bar{x} + b，方差 = a²s²
平均数：ȳ = (1/n)∑(axᵢ+b) = a·(1/n)∑xᵢ + (1/n)·nb = a\bar{x} + b
方差：s²(y) = (1/n)∑(yᵢ - ȳ)² = (1/n)∑[(axᵢ+b) - (a\bar{x}+b)]²
= (1/n)∑[a(xᵢ - \bar{x})]² = a²·(1/n)∑(xᵢ - \bar{x})² = a²s²
结论：线性变换 y = ax + b 下，
平均数：加上b（平移），乘上a（缩放）
方差：只被a影响（缩放），不受b影响（平移不变性），且a被平方
记忆口诀：平均数跟着变，方差只受乘影响，平移不改变分散程度。
拔高4加权平均数
某次考试，甲班50人平均分为78分，乙班40人平均分为82分，丙班30人平均分为85分。求三个班的总平均分。（保留2位小数）
答案：81.33分
不能直接加三个平均分除以3（78+82+85)/3≈81.67——❌ 这是错的！
因为各班人数不同，需要用加权平均数：
总平均分 = (甲总分 + 乙总分 + 丙总分) / 总人数
甲总分 = 50 × 78 = 3900
乙总分 = 40 × 82 = 3280
丙总分 = 30 × 85 = 2550
总平均分 = (3900 + 3280 + 2550) / (50 + 40 + 30) = 9760 / 120 ≈ 81.33分
易错点：各组人数不同时，简单平均(算术平均)会出错，必须用加权平均。
拔高5样本与总体
某公司有员工200人，其中研发部80人、市场部60人、后勤部60人。为调查员工满意度，分别从三个部门用简单随机抽样各抽取10人，共30人。请问这是否是分层抽样？理由是什么？
答案：不是严格的分层抽样
分层抽样的核心要求：各层抽取比例相同，即按人数比例分配样本容量。
三个部门各抽10人，总人数分别为80,60,60。
研发部比例：10/80 = 1/8 = 12.5%
市场部比例：10/60 = 1/6 ≈ 16.7%
后勤部比例：10/60 = 1/6 ≈ 16.7%
各层比例不同，所以这不是严格意义的分层抽样。
如果改为按比例：总200人样本30人，比例k=30/200=3/20，
则研发应抽80×(3/20)=12人，市场60×(3/20)=9人，后勤60×(3/20)=9人。
本题的抽样方法实际上是"分层后等额抽样"（每层抽相同数量），
在某些情况下也会使用（如比较各层差异），但比例的代表性不如真正的分层抽样。
拔高6百分位数应用
某公司员工月薪（单位：千元）如下：
3, 4, 4, 5, 5, 5, 6, 6, 7, 8, 8, 9, 10, 12, 15, 20, 25, 30, 35, 50。
(1) 求这组数据的第90百分位数。
(2) 若公司声称"90%的员工月薪不超过某个数"，这个数是多少？
答案：(1) 32.5千元   (2) 32.5千元
数据已从小到大排列，共20个数据。
(1) i = n × p% = 20 × 90% = 18
i=18是整数，取第18个和第19个的平均值。
第18个数：30，第19个数：35
第90百分位数 = (30+35)/2 = 32.5（千元）。
(2)"90%的员工月薪不超过某个数"——这个数就是第90百分位数。
即 32.5千元 = 32500元。意味着有90%的员工月薪低于或等于32500元。
注意：百分位数的实际意义——p%的数据小于等于第p百分位数。
拔高7标准差综合
甲、乙两组各5名学生的数学成绩如下：
甲组：70, 75, 80, 85, 90
乙组：60, 70, 80, 90, 100
(1) 分别计算两组数据的平均数和标准差；
(2) 哪组数据的离散程度更大？为什么？
答案：(1) 甲：\bar{x}=80, s≈7.07；乙：\bar{x}=80, s≈14.14 (2) 乙组更大
(1) 甲组：
平均数 = (70+75+80+85+90)/5 = 400/5 = 80
方差（总体）= [(70-80)²+(75-80)²+0²+5²+10²]/5 = [100+25+0+25+100]/5 = 250/5 = 50
标准差 = √50 = 5√2 ≈ 7.07
乙组：
平均数 = (60+70+80+90+100)/5 = 400/5 = 80
方差 = [(-20)²+(-10)²+0²+10²+20²]/5 = [400+100+0+100+400]/5 = 1000/5 = 200
标准差 = √200 = 10√2 ≈ 14.14
(2) 两组平均数相同（都是80），但乙组的标准差更大（14.14 > 7.07），
说明乙组的成绩离散程度更大，即数据更分散、两极分化更明显。
直观理解：甲组从70到90均匀分布，乙组从60到100分布更广。
拔高8抽样方法辨析
以下抽样方法是否合理？如不合理，请说明理由并提出改进建议。
(1) 为了调查某路口车流量，只统计周一上午8:00-9:00的数据。
(2) 为了了解全校学生的视力情况，在操场上随机询问正在上体育课的学生。
(3) 为了了解某小区居民对垃圾分类的态度，在小区业主群发放电子问卷。
答案：三个都不合理
(1) 不合理。
理由：只统计周一早高峰，不能代表一周平均车流量（周末、非高峰时段流量不同）。
建议：在一周不同日期、不同时段（早高峰、平峰、晚高峰）分别抽样，或采用等距时间抽样。
(2) 不合理。
理由：上体育课的学生身体活动能力强，视力可能相对较好，存在样本偏差（健康样本偏差）。
建议：按学号用简单随机抽样或分层抽样（按年级层）从全校学生中抽取。
(3) 不合理。
理由：业主群只能覆盖会使用手机的老人和年轻人，且愿意填问卷的居民可能态度更积极，
存在覆盖偏差和自愿回答偏差。
建议：结合入户访问和线上问卷相结合，或采用分层抽样（按年龄层、楼栋层）。
核心：样本要代表总体，避免系统性偏差。
拔高9统计图表综合
某班50名学生期中考试数学成绩的频率分布直方图中，
各组的频率/组距值如下（组距均为10）：
[30,40)：0.005   [40,50)：0.010   [50,60)：0.020
[60,70)：0.030   [70,80)：0.025   [80,90)：0.010   [90,100]：0.005
(1) 求成绩在[60,80)的人数；
(2) 求该班数学成绩的平均分估计值；
(3) 该直方图是否合理？说明理由。
答案：(1) 27.5人≈28人   (2) 68.75分   (3) 不合理
(1) [60,70)频率 = 0.030×10 = 0.30
[70,80)频率 = 0.025×10 = 0.25
[60,80)频率 = 0.30+0.25 = 0.55
人数 = 50×0.55 = 27.5人（实际应为整数，约28人）。
(2) 用各组组中值估计：
30-40：35×0.005×10 = 35×0.05 = 1.75
40-50：45×0.010×10 = 45×0.1 = 4.5
50-60：55×0.020×10 = 55×0.2 = 11
60-70：65×0.030×10 = 65×0.3 = 19.5
70-80：75×0.025×10 = 75×0.25 = 18.75
80-90：85×0.010×10 = 85×0.1 = 8.5
90-100：95×0.005×10 = 95×0.05 = 4.75
平均分 = 1.75+4.5+11+19.5+18.75+8.5+4.75 = 68.75分
(3) 不合理。
检查频率之和：0.05+0.1+0.2+0.3+0.25+0.1+0.05 = 1.05 > 1
频率之和不应大于1！说明直方图中各组频率/组距数据有误。
应调整为总和正好等于1。例如可能[60,70)的频密度应为0.025而不是0.030。
易错点：频率分布直方图中，各小长方形面积之和必须等于1。
