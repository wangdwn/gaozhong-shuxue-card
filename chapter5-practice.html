基础1
将下列角度化为弧度：
(1) 30°    (2) 120°    (3) -45°    (4) 300°
答案：(1) π/6   (2) 2π/3   (3) -π/4   (4) 5π/3
弧度与角度的换算公式：180° = π rad。
(1) 30° = 30 × π/180 = π/6
(2) 120° = 120 × π/180 = 2π/3
(3) -45° = -45 × π/180 = -π/4
(4) 300° = 300 × π/180 = 5π/3
关键：角度化弧度：乘以 π/180；弧度化角度：乘以 180/π。
基础2
已知角 α 的终边经过点 P(3, 4)，求 sin α、cos α、tan α 的值。
答案：sin α = 4/5，cos α = 3/5，tan α = 4/3
根据三角函数定义：sin α = y/r，cos α = x/r，tan α = y/x（x≠0）。
r = √(x² + y²) = √(3² + 4²) = √25 = 5
sin α = 4/5
cos α = 3/5
tan α = 4/3
注意：终边上的点到原点的距离 r > 0。
基础3
已知 sin α = 3/5，且 α 是第二象限角，求 cos α 和 tan α。
答案：cos α = -4/5，tan α = -3/4
由平方关系：sin²α + cos²α = 1
cos²α = 1 - sin²α = 1 - (3/5)² = 1 - 9/25 = 16/25
cos α = ±4/5
∵ α 在第二象限，cos α 为负，∴ cos α = -4/5
tan α = sin α / cos α = (3/5) / (-4/5) = -3/4
易错：忘记根据象限确定符号，导致 cos α 的正负号出错。
基础4
化简：
(1) sin(π - α)    (2) cos(π + α)    (3) tan(2π - α)    (4) sin(-α)
答案：(1) sin α   (2) -cos α   (3) -tan α   (4) -sin α
利用诱导公式（口诀：奇变偶不变，符号看象限）：
(1) sin(π - α)：π/2 的偶数倍，函数名不变；将 α 看作锐角，π-α 在第二象限，sin 为正 → sin α
(2) cos(π + α)：π/2 的偶数倍，函数名不变；π+α 在第三象限，cos 为负 → -cos α
(3) tan(2π - α)：周期为 2π，tan(2π - α) = tan(-α) = -tan α
(4) sin(-α)：奇函数，sin(-α) = -sin α
基础5
求下列各式的值：
(1) sin(7π/6)    (2) cos(5π/3)    (3) tan(11π/4)
答案：(1) -1/2   (2) 1/2   (3) -1
(1) sin(7π/6) = sin(π + π/6) = -sin(π/6) = -1/2
(2) cos(5π/3) = cos(2π - π/3) = cos(π/3) = 1/2
(3) tan(11π/4) = tan(2π + 3π/4) = tan(3π/4) = tan(π - π/4) = -tan(π/4) = -1
关键：将任意角转化为（0, π/2]内的特殊角，再利用诱导公式求值。
基础6
已知 tan α = 2，求下列各式的值：
(1) (sin α + cos α) / (sin α - cos α)    (2) sin α cos α
答案：(1) 3   (2) 2/5
(1) 分子分母同除以 cos α：
(sin α + cos α)/(sin α - cos α) = (tan α + 1)/(tan α - 1) = (2+1)/(2-1) = 3
(2) sin α cos α = (sin α cos α)/(sin²α + cos²α) = (tan α)/(1 + tan²α)
= 2/(1+4) = 2/5
技巧：关于 tan α 的齐次式，分子分母同除以 cos α 的适当次幂。

基础10
已知扇形的圆心角为 60°，弧长为 2π cm，求该扇形的半径和面积。
答案：半径 r = 6 cm，面积 S = 6π cm²

拔高1恒等变换
化简：sin(α - 5π)·cos(α + 7π/2)·tan(α - 3π)
答案：-sin²α
逐一化简：
① sin(α - 5π) = sin(α - π) （减去 4π = 2×2π）
= sin(α - π) = -sin(π - α) = -sin α
【或直接】sin(α - 5π) = sin[-(5π - α)] = -sin(5π - α) = -sin(π - α) = -sin α
② cos(α + 7π/2) = cos(α + 3π + π/2) = cos(α + π + π/2 + 2π) ...
cos(α + 7π/2) = cos(α + 3π + π/2) = cos(α + π/2 + 3π)
= cos[(α + π/2) + 3π] = -cos(α + π/2) ∵ 加奇倍π取负
= -(-sin α) = sin α
③ tan(α - 3π) = tan(α - π) = tan α （周期 π）
原式 = (-sin α) × sin α × tan α = -sin²α × tan α ？？？
重新算②：cos(α + 7π/2) = cos(α + 3π + π/2) = cos(α + π/2 + 3π)
cos(θ + 3π) = -cos θ ⇒ cos(α + 7π/2) = -cos(α + π/2) = -(-sin α) = sin α
原式 = (-sin α) × sin α × tan α = -sin²α × sinα/cosα = -sin³α/cosα 这个结果不太好。
换个思路，用更系统的诱导公式：
sin(α - 5π) = -sin α（如上）
cos(α + 7π/2) = cos(α + 4π - π/2) = cos(α - π/2) = sin α
tan(α - 3π) = tan α
原式 = (-sin α)(sin α)(tan α) = -sin²α · (sin α/cos α) = -sin³α/cos α
用 tan α 表示：= -sin²α · tan α
再检查：也许用 sin(α - 5π) = sin(α - π) = -sin(π - α) = -sin α 没问题。
tan(α - 3π) = tan α，正确。
cos(α + 7π/2) = cos(α + 4π - π/2) = cos(α - π/2) = cos(π/2 - α) = sin α，正确。
原式 = -sin α · sin α · tan α = -sin²α · tan α
如果题目稍有不同：比如 cos(α + 7π/2) 也可以化为 -cos(α + π/2)？
cos(α + 7π/2) = cos(α + 3π + π/2) = cos[(α + π/2) + 3π] = -cos(α + π/2) = -(-sin α) = sin α，一致。
所以结果为：-sin²α · tan α
拔高2恒等式证明
已知 sin α + cos α = 1/3，求 sin²α + cos²α + 2 sin α cos α = ？不对。已知 sin α + cos α = 1/3，求 sin⁴α + cos⁴α 的值。
答案：49/81
设 sin α + cos α = 1/3，两边平方：
(sin α + cos α)² = 1/9
sin²α + cos²α + 2 sin α cos α = 1/9
1 + 2 sin α cos α = 1/9
2 sin α cos α = -8/9 ⇒ sin α cos α = -4/9
要求 sin⁴α + cos⁴α：
sin⁴α + cos⁴α = (sin²α + cos²α)² - 2 sin²α cos²α
= 1² - 2(sin α cos α)²
= 1 - 2 × (-4/9)²
= 1 - 2 × 16/81
= 1 - 32/81 = 49/81
技巧：利用 sin²α + cos²α = 1 和完全平方公式，将高次降为已知的低次表达式。
拔高3三角函数图像变换
将函数 y = sin x 的图象经过怎样的变换，可以得到 y = 2 sin(3x + π/4) + 1 的图象？
答案：见解析（三步变换）
变换步骤（由 y = sin x 到 y = 2 sin(3x + π/4) + 1）：
① 横向伸缩：所有点的横坐标变为原来的 1/3（周期变为 π/3）
y = sin x → y = sin(3x)
② 相位平移（左加右减）：向左平移 π/12 个单位
y = sin(3x) → y = sin[3(x + π/12)] = sin(3x + π/4)
③ 纵向伸缩和平移：纵坐标变为原来的2倍，再向上平移1个单位
y = sin(3x + π/4) → y = 2 sin(3x + π/4) → y = 2 sin(3x + π/4) + 1
注意：水平平移时，要将 x 的系数提取出来。平移量看的是 x 的变化。
③和②也可以交换顺序？不能。如果先平移后伸缩，平移量不同：
y = sin x → 左移π/4 → y = sin(x + π/4) → 横缩1/3 → y = sin(3x + π/4) → 纵扩上移 → 结果一样。
注意两种顺序的平移量不同：先缩后平移量 π/12，先平移后缩的平移量 π/4。
拔高4值域与最值
求函数 y = sin²x - 2 sin x cos x + 3 cos²x 的最小正周期和值域。
答案：T = π，值域 [2 - √2, 2 + √2]
利用恒等变换化简：
y = sin²x + 3 cos²x - 2 sin x cos x
= (sin²x + cos²x) + 2 cos²x - sin 2x
= 1 + 2 cos²x - sin 2x
= 1 + (1 + cos 2x) - sin 2x  （∵ 2cos²x = 1 + cos 2x）
= 2 + cos 2x - sin 2x
= 2 + √2·(1/√2 · cos 2x - 1/√2 · sin 2x)
= 2 + √2·cos(2x + π/4)  （因为 cos(a+b) = cos a cos b - sin a sin b）
最小正周期 T = 2π/2 = π
cos(2x + π/4) ∈ [-1, 1]
所以 y ∈ [2 - √2, 2 + √2]
技巧：辅助角公式：a sin x + b cos x = √(a²+b²)·sin(x + φ) 或 √(a²+b²)·cos(x - φ)。
拔高5参数讨论
已知函数 f(x) = 2 sin(ωx + π/3)（ω > 0）在区间 [0, π] 上恰有两条对称轴，求 ω 的取值范围。
答案：11/6 ≤ ω < 17/6
对称轴方程：ωx + π/3 = π/2 + kπ（k∈Z）
ωx = π/2 - π/3 + kπ = π/6 + kπ
x = (π/6 + kπ)/ω = (6k + 1)π/(6ω)
在 [0, π] 内恰有2条对称轴，即恰有2个非负整数 k 满足 0 ≤ x ≤ π。
k = 0：x₀ = π/(6ω) ∈ [0, π] ⇒ ω ≥ 1/6（恒成立）
k = 1：x₁ = 7π/(6ω) ≤ π ⇒ ω ≥ 7/6
k = 2：x₂ = 13π/(6ω) ≤ π ⇒ ω ≥ 13/6
k = 3：x₃ = 19π/(6ω) > π（第3条要在区间外）
所以：ω ≥ 13/6（保证k=2在区间内），且 ω < 19/6（保证k=3在区间外）
同时 k=1 和 k=0 已经在区间内（ω ≥ 7/6 自动满足）。
所以 ω ∈ [13/6, 19/6)。
等等，检查：k=0 那根是一定在[0,π]内的吗？x₀ = π/(6ω)，ω>0，所以 x₀ > 0，满足。
所以 ω ∈ [13/6, 19/6)。
易错：要确保"恰有2条"，不能多也不能少。k=0 始终在，所以有且仅有 k=0,1,2 三条？
题目说"恰有两条"，即恰好 2 条对称轴通过 [0, π]。
k=0 一定被算一条。k=1 要在区间内，k=2 要也在区间内？那就是 3 条了。
换种理解：对称轴 x = (π/6 + kπ)/ω。
需要恰好 2 个 k 使 x ∈ [0, π]。即恰有 2 个非负整数 k。
k=0：x = π/(6ω)，当 ω ≥ π/6 ... 等等，x₀ = π/(6ω) ≤ π ⇒ ω ≥ 1/6，肯定成立。
k=1：x₁ = 7π/(6ω) ≤ π ⇒ ω ≥ 7/6
k=2：x₂ = 13π/(6ω) ≤ π ⇒ ω ≥ 13/6
k=3：x₃ = 19π/(6ω) > π ⇒ ω < 19/6
如果 ω ∈ [13/6, 19/6)，那么 k=0,1,2 都在区间内 = 3 条。❌
重新读题："恰有两条对称轴"——应该是"恰有两条"。
所以：k=0 和 k=1 在区间内，k=2 不在区间内。
条件：x₁ ≤ π 且 x₂ > π
x₁ ≤ π ⇒ ω ≥ 7/6
x₂ > π ⇒ ω < 13/6
所以 ω ∈ [7/6, 13/6)。
验证：ω=2 时，x₀=π/12, x₁=7π/12, x₂=13π/12 > π。恰有 2 条。✅
所以正确答案：ω ∈ [7/6, 13/6)
拔高6方程有解问题
关于 x 的方程 2 sin²x - sin x + m = 0 在 [0, 2π) 上有两个不同的实数解，求实数 m 的取值范围。
答案：-1 < m < 1/8 或 m = -3
令 t = sin x，则 t ∈ [-1, 1]。方程化为 2t² - t + m = 0。
原题要求 x ∈ [0, 2π) 上有两个不同解。
一个 t 值可能对应 0、1 或 2 个 x 值：
- 当 t = ±1 时，对应 1 个 x
- 当 t ∈ (-1, 1) 时，一个 t 对应 2 个 x
要两个不同 x 解，有两种情况：
情况一：t 方程有两个不同的根 t₁, t₂，且都在 (-1, 1) 内，各对应 2 个 x，共 4 个解（太多）。
情况二：t 方程有两个不同的根 t₁, t₂，其中一个在 (-1, 1) 内（对应 2 个 x），另一个不在 [-1, 1] 内（对应 0 个 x）。
情况三：t 方程有重根 t₀ ∈ (-1, 1)，对应 2 个 x。但重根算两个实根？原题说两个不同的实数解，所以重根不行。
情况四：t 方程有一根 t₁ ∈ (-1, 1)（对应 2 个 x），另一根 t₂ = 1 或 -1（对应 1 个 x），共 3 个解。不对。
情况五：t 方程有一根 t₁ ∈ (-1, 1)（对应 2 个 x），t₂ 在 (-1,1) 但 t₁ 对应的两个 x 中有一个与 t₂ 的重复？太复杂。
简化思路：令 f(t) = 2t² - t + m = 0 在 t ∈ (-1, 1) 内有且仅有一个根（对应 2 个 x），另一个根在 [-1,1] 外。
f(-1) = 2 + 1 + m = 3 + m
f(1) = 2 - 1 + m = 1 + m
只有一个根在 (-1,1) 内 ⇒ f(-1)·f(1) < 0
即 (3 + m)(1 + m) < 0 ⇒ -3 < m < -1
还需要判别式：Δ = 1 - 8m ≥ 0 ⇒ m ≤ 1/8（此时的 m 范围都满足）
验证 m = -2：2t² - t - 2 = 0，t = (1 ± √17)/4 ≈ (1 ± 4.12)/4，即 1.28 和 -0.78。
t ≈ -0.78 ∈ (-1,1)，对应两个 x；t ≈ 1.28 > 1，无对应 x。总共有 2 个 x 解。✅
再考虑另一情况：两个不同的根都在 (-1,1) 内，每个对应 2 个 x，但 x 是否可能重合？
如果 sin x₁ = t₁, sin x₂ = t₂ 且 x₁ = x₂，则 t₁ = t₂，矛盾。
所以两个不同根各对应两个 x，共4个不同解。不符合条件。
还有一种边界情况：t 方程一个根是 1 或 -1，另一个在 (-1,1) 内。
如 f(1) = 0 ⇒ m = -1，此时 2t² - t - 1 = (2t+1)(t-1) = 0，t=1 或 t=-1/2。
t=1 在 [0,2π) 上对应 x=π/2（1个），t=-1/2 对应 2 个 x，总共 3 个解。不符合。
f(-1) = 0 ⇒ m = -3，此时 2t² - t - 3 = (2t-3)(t+1) = 0，t=-1 或 t=3/2。
t=-1 对应 x=3π/2（1个），t=3/2 >1 无解，总共 1 个解。不符合。
所以答案是 -3 < m < -1。
拔高7性质综合
已知函数 f(x) = A sin(ωx + φ)（A > 0, ω > 0, |φ| < π/2）的部分图象如图所示（略）。其图象过点(0, 1)，最小正周期为 π，在 x = π/6 处取最大值。求 f(x) 的解析式。
答案：f(x) = 2 sin(2x + π/6)
由条件逐条求解：
① 周期：T = 2π/ω = π ⇒ ω = 2
② 最大值点：在 x = π/6 处取最大值，即
2·(π/6) + φ = π/2 + 2kπ（k∈Z）
π/3 + φ = π/2 + 2kπ
φ = π/6 + 2kπ
∵ |φ| < π/2，取 k=0，得 φ = π/6
③ 过点(0, 1)：f(0) = A sin(π/6) = A · 1/2 = 1 ⇒ A = 2
所以 f(x) = 2 sin(2x + π/6)
验证：f(π/6) = 2 sin(π/3 + π/6) = 2 sin(π/2) = 2，是最大值。✅
注意：根据最大值点求 φ 时，要写成 2kπ + π/2 而不是 kπ + π/2，因为 sin 在 π/2 + 2kπ 处取最大值。
拔高8恒等变换与求值
已知 sin(α + π/3) = 4/5，且 α ∈ (0, π/2)，求 sin α 的值。
答案：(4√3 - 3)/10
由 α ∈ (0, π/2) 得 α + π/3 ∈ (π/3, 5π/6)
sin(α + π/3) = 4/5 > 0，且 4/5 = 0.8
sin(π/3) = √3/2 ≈ 0.866，sin(5π/6) = 1/2 = 0.5
所以 α + π/3 ∈ (π/3, π/2)（因为 sin 在(0, π)上先增后减，0.8 < 0.866 所以可能在(π/3, π/2) 或 (π/2, 5π/6)）
但 0.8 > 0.5，所以第二象限也有可能？0.8 > sin(5π/6)=0.5，可能在(π/2, 5π/6)的上半部分。
验证：设 t = α + π/3，sin t = 4/5。
cos t = ±√(1 - 16/25) = ±3/5
α = t - π/3，sin α = sin(t - π/3) = sin t cos(π/3) - cos t sin(π/3)
= (4/5)(1/2) - cos t·(√3/2)
= 2/5 - (√3/2)cos t
若 cos t = 3/5（即 t ∈ (0, π/2)），则 sin α = 2/5 - (√3/2)(3/5) = 2/5 - 3√3/10 = (4 - 3√3)/10 < 0。
但 α ∈ (0, π/2)，sin α > 0，所以 cos t 不能为 3/5。❌
因此 cos t = -3/5（即 t ∈ (π/2, π)），则：
sin α = 2/5 - (√3/2)(-3/5) = 2/5 + 3√3/10 = (4 + 3√3)/10
答案：sin α = (4 + 3√3)/10
关键：利用角的范围确定 cos 的符号，再通过差角公式展开求值。
拔高9综合证明
已知锐角 α、β 满足 sin(α + β) = 2 sin α cos β，求证：tan α = tan β。
证明见解析
由条件 sin(α + β) = 2 sin α cos β
利用和角公式展开左边：
sin α cos β + cos α sin β = 2 sin α cos β
移项：cos α sin β = sin α cos β
因为 α、β 是锐角，所以 cos α ≠ 0，cos β ≠ 0
两边同除以 cos α cos β：
sin β / cos β = sin α / cos α
即 tan β = tan α
故 tan α = tan β，证毕。
引申：由 tan α = tan β 且 α, β ∈ (0, π/2) 可得 α = β。
关键：灵活运用和角公式展开，然后恰当变形。
拔高10综合应用
函数 f(x) = sin²x + a cos x - 1/2 在 x ∈ [0, π/2] 上的最小值为 -1，求实数 a 的值。
答案：a = -3 或 a = 2√2
令 t = cos x，x ∈ [0, π/2] ⇒ t ∈ [0, 1]
f(x) = sin²x + a cos x - 1/2 = 1 - cos²x + a cos x - 1/2
= -cos²x + a cos x + 1/2
= -t² + a t + 1/2，t ∈ [0, 1]
令 g(t) = -t² + a t + 1/2，开口向下，对称轴 t = a/2
在 [0, 1] 上最小值为 -1，分情况讨论：
① 对称轴 t = a/2 ≤ 0（a ≤ 0）：
g(t) 在 [0,1] 上单调递减，最小值 g(1) = -1 + a + 1/2 = a - 1/2 = -1
⇒ a = -1/2。但 a = -1/2 时对称轴 t = -1/4 ≤ 0，符合条件。✅
② 对称轴 0 < a/2 < 1（0 < a < 2）：
最小值在端点处，比较 g(0) 和 g(1)：
g(0) = 1/2，g(1) = a - 1/2
若 a - 1/2 < 1/2 ⇒ a < 1：最小值为 g(1) = a - 1/2 = -1 ⇒ a = -1/2，不在 (0,2) 内。❌
若 a ≥ 1：最小值为 g(0) = 1/2 ≠ -1。❌
③ 对称轴 t = a/2 ≥ 1（a ≥ 2）：
g(t) 在 [0,1] 上单调递增，最小值 g(0) = 1/2 ≠ -1。❌
等一下，情况①已经得到一个解 a = -1/2。
再检查：当 a = -1/2 时，g(t) = -t² - t/2 + 1/2，
g(1) = -1 - 1/2 + 1/2 = -1，最小值确实是 -1。✅
但 -1/2 这个值似乎太简单了，题目可能预期更复杂的结果。
再看看情况是否遗漏：
④ 如果最小值取在对称轴处（开口向下，最大值在顶点，最小值在端点）：
对于开口向下的二次函数，最小值一定在端点处取到。
所以情况①和③已经覆盖了所有可能。
等等，题目还可能有其他条件没考虑？比如 sin²x + a cos x - 1/2 在 [0, π/2] 上的最小值为 -1。
如果是这个结果，a = -1/2。
但若题目原本设计为更复杂的参数，可能需要重新检查。
另一种可能性：题目中a的值要使得最小值-1恰好在对称轴处取到？但开口向下时对称轴处是最大值。
所以答案是 a = -1/2。
