基础1
下列命题中正确的是（ ）
(A) 零向量没有方向
(B) 若两个向量相等，则它们的起点和终点分别重合
(C) 若向量 a 与 b 共线，则 a 与 b 的方向相同或相反
(D) 单位向量都相等
答案：C
选项A：零向量的方向是任意的，并非没有方向，错误。
选项B：向量相等只要求大小相等且方向相同，与起点终点位置无关（自由向量），错误。
选项C：对。共线向量即方向相同或相反（含零向量），正确。
选项D：单位向量是模长等于1的向量，但方向可以不同，不一定相等。错误。
易错提醒：向量相等的定义——大小相等且方向相同，与位置无关。
基础2
在平行四边形 ABCD 中，AB = a，AD = b，则 AC 和 DB 分别等于什么？
答案：AC = a + b，DB = a − b
平行四边形法则：对角线向量 AC 等于相邻两边向量之和。
即 AC = AB + AD（首尾相接） = a + b。
DB = AB − AD（从D指向B）= a − b。
或者：DB = AB − AD，因为 DB = DA + AB = −b + a。
基础3
已知向量 a = (3, −1)，b = (−1, 2)，求 a + b，2a − 3b 的坐标。
答案：a+b = (2, 1)，2a−3b = (9, −8)
加法：a + b = (3+(−1), −1+2) = (2, 1)
数乘与加减：2a − 3b = 2(3, −1) − 3(−1, 2)
= (6, −2) − (−3, 6) = (6−(−3), −2−6) = (9, −8)
注意：向量坐标运算就是对应坐标分别计算。
基础4
已知向量 a = (1, 2)，b = (x, 4)，且 a ∥ b，求 x 的值。
答案：x = 2
两向量平行（共线）的条件是对应坐标成比例：1/x = 2/4
即 2x = 4 × 1？注意交错相乘：1×4 = 2×x
4 = 2x ⇒ x = 2
一般公式：a∥b (b≠0) ⇔ x₁y₂ = x₂y₁。
验证：a = (1,2)，b = (2,4) = 2(1,2) = 2a，确实平行。
基础5
已知 a = (2, 1)，b = (3, 4)，求 a·b 及向量 a 与 b 的夹角 θ 的余弦值。
答案：a·b = 10，cosθ = 2√5/5
数量积（点积）：a·b = x₁x₂ + y₁y₂ = 2×3 + 1×4 = 6 + 4 = 10
模长：|a| = √(2²+1²) = √5，|b| = √(3²+4²) = 5
cosθ = (a·b) / (|a|·|b|) = 10 / (√5 × 5) = 10/(5√5) = 2/√5 = 2√5/5
注意：数量积的结果是一个实数，不是向量。
基础6
已知向量 a = (1, 0)，b = (1, 1)。求 a 在 b 方向上的投影数量。
答案：√2/2
投影公式：a 在 b 方向上的投影为 (a·b) / |b|
a·b = 1×1 + 0×1 = 1
|b| = √(1²+1²) = √2
投影 = 1/√2 = √2/2
注意区别：投影数量是标量（可正可负）；投影向量是向量。
基础7
设 a = (2, 3)，b = (−4, 6)。判断 a 与 b 是垂直、平行还是其它关系？
答案：垂直
先检查平行：2/(−4) = −1/2，3/6 = 1/2。比值不相等，不平行。
再检查垂直：数量积 a·b = 2×(−4) + 3×6 = −8 + 18 = 10 ≠ 0
咦？数量积不为0，说明也不垂直？让我重新计算……
等等，再算一次：a = (2, 3)，b = (−4, 6)
a·b = 2×(−4) + 3×6 = −8 + 18 = 10 ≠ 0，既不平行也不垂直。
结论：两个向量既不平行也不垂直，是一般关系。
易错警示：不要看到坐标有倍数关系就认为平行，需验证对应坐标成比例。
基础8
已知向量 a = (2, −2)，b = (k, −5)，且 a⊥b，求实数 k 的值。
答案：k = −5
两向量垂直的条件：数量积为 0。
a·b = 2×k + (−2)×(−5) = 2k + 10
令 2k + 10 = 0 ⇒ k = −5
验证：a = (2, −2)，b = (−5, −5)
a·b = 2×(−5) + (−2)×(−5) = −10 + 10 = 0 ✅
基础9
已知 A(1, 2)，B(3, 0)，C(5, 6)。若四边形 ABCD 为平行四边形，求顶点 D 的坐标。
答案：D(7, 4)
平行四边形的对角线互相平分，即对角线交点为中点。
设 D(x, y)。由 AB = DC（一组对边平行且相等）。
AB = (3−1, 0−2) = (2, −2)
DC = (5−x, 6−y)
令 AB = DC：2 = 5−x ⇒ x=3；−2 = 6−y ⇒ y=8
得 D(3, 8)。等等，再检查另一种对应方式。
方法一（对边平行）：AD = BC
BC = (5−3, 6−0) = (2, 6)，AD = (x−1, y−2)
x−1=2 ⇒ x=3；y−2=6 ⇒ y=8 → D(3, 8)
方法二（对角线中点相同）：AC中点 = BD中点
AC中点M = ((1+5)/2, (2+6)/2) = (3, 4)
BD中点 = ((3+x)/2, (0+y)/2) = (3, 4)
(3+x)/2=3 ⇒ x=3；(0+y)/2=4 ⇒ y=8 → D(3, 8)
最终答案：D(3, 8)
基础10
已知 a = (1, 2)，b = (2, −1)，c = (3, 4)。若 c = λa + μb，求 λ 和 μ 的值。
答案：λ = 11/5，μ = 2/5
λa + μb = λ(1, 2) + μ(2, −1) = (λ+2μ, 2λ−μ)
令其等于 c = (3, 4)，得方程组：
① λ + 2μ = 3
② 2λ − μ = 4
由②得：μ = 2λ − 4
代入①：λ + 2(2λ−4) = 3 ⇒ λ + 4λ − 8 = 3 ⇒ 5λ = 11 ⇒ λ = 11/5
μ = 2×(11/5) − 4 = 22/5 − 20/5 = 2/5
即：c = (11/5)a + (2/5)b
拔高1向量分解
在△ABC中，AB = c，AC = b。点 D 在边 BC 上，且 BD = 2DC。试用 b，c 表示向量 AD。
答案：AD = (1/3)b + (2/3)c
BC = AC − AB = b − c
BD = 2DC ⇒ D 分 BC 为 2:1，所以 BD = (2/3)BC = (2/3)(b − c)
则 AD = AB + BD = c + (2/3)(b − c)
= c + (2/3)b − (2/3)c = (2/3)b + (1/3)c
验证：也可用定比分点公式直接得 AD = (1×AB + 2×AC)/(1+2) = (c+2b)/3
即 AD = (1/3)b + (2/3)c。注意：这里与上面的顺序相反？
纠正：定比分点公式：D分BC，BD:DC=2:1 ⇒ AD = (AB×1 + AC×2)/(1+2) = (c + 2b)/3 = (2/3)b + (1/3)c
最终：AD = (2/3)b + (1/3)c
拔高2共线问题
设两个非零向量 a 与 b 不共线。
(1) 若 AB = a + b，BC = 2a + 8b，CD = 3(a − b)。求证：A, B, D 三点共线。
(2) 试确定实数 k，使 ka + b 与 a + kb 共线。
答案：(1) 见证明 (2) k = ±1
(1) 证明三点共线：
要证 A, B, D 共线，即证 AB 与 BD 共线（或 AB 与 AD 共线）。
先求 BD = BC + CD = (2a+8b) + 3(a−b) = 5a + 5b = 5(a+b) = 5AB
所以 BD = 5AB，说明 BD 与 AB 共线（有公共点B），故 A, B, D 共线。✅
(2) 共线条件：存在实数 λ 使得 ka+b = λ(a+kb)
整理：(k−λ)a + (1−λk)b = 0
由于 a, b 不共线 ⇒ 系数全为零：
① k − λ = 0 ⇒ λ = k
② 1 − λk = 0 ⇒ 1 − k² = 0 ⇒ k² = 1 ⇒ k = ±1
注意：当 k=1 时，两向量相等；当 k=−1 时，两向量相反。都满足共线。
拔高3数量积最值
已知向量 a = (3, 4)，b = (−1, 2)。若向量 c 满足 c ⊥ a 且 |b − c| 取最小值，求 c 的坐标。
答案：c = (8/5, −6/5)
设 c = (x, y)。由 c⊥a 得：3x + 4y = 0 ⇒ y = −(3/4)x
则 |b − c|² = |(−1−x, 2−y)|² = (−1−x)² + (2−y)²
代入 y = −(3/4)x：
= (−1−x)² + (2 + 3x/4)²
= (1 + 2x + x²) + (4 + 3x + 9x²/16)
= x² + 9x²/16 + 2x + 3x + 1 + 4
= (25/16)x² + 5x + 5
这是关于 x 的二次函数，开口向上，最小值在顶点处：
x = −5 / (2 × 25/16) = −5 / (25/8) = −5 × 8/25 = −8/5
此时 y = −(3/4)×(−8/5) = 6/5
所以 c = (−8/5, 6/5)
几何意义：b−c 取最小值时，c 是 b 在垂直于 a 的方向上的投影。
拔高4向量与几何
在△ABC中，O是△ABC的重心，G是△ABC的重心（中线交点）。求证：对于任意一点P，有 PA + PB + PC = 3PG。
证明见解析
PA = PG + GA
PB = PG + GB
PC = PG + GC
三式相加：PA + PB + PC = 3PG + (GA + GB + GC)
由于G是重心，有 GA + GB + GC = 0（重心性质）
所以 PA + PB + PC = 3PG。证毕。✅
推广：若G是重心，则 PG = (PA + PB + PC)/3，即重心坐标公式。
特别地，取P为原点O，得 OG = (OA + OB + OC)/3。
拔高5应用问题
一物体受到两个力 F₁ = (3, 2) 牛顿和 F₂ = (1, −4) 牛顿的作用。求合力的大小和方向（与x轴正方向的夹角的正切值）。
答案：合力大小 2√5 N，tanθ = −1/2
合力 F = F₁ + F₂ = (3+1, 2+(−4)) = (4, −2)
合力大小：|F| = √(4² + (−2)²) = √(16+4) = √20 = 2√5 (N)
设合力与x轴正方向夹角为θ，则 tanθ = (−2)/4 = −1/2
因为 x>0 且 y<0，所以θ在第四象限，θ = arctan(−1/2)（负角）。
物理意义：向量加法在物理中的直接应用——力的合成。
拔高6向量求模
已知向量 a，b 满足 |a| = 2，|b| = 3，a 与 b 的夹角为60°。
(1) 求 |a + b|；
(2) 求 |a − 2b|。
答案：(1) √19 (2) 2√7
(1) |a+b|² = (a+b)·(a+b) = |a|² + 2a·b + |b|²
a·b = |a||b|cos60° = 2×3×(1/2) = 3
所以 |a+b|² = 4 + 2×3 + 9 = 4 + 6 + 9 = 19
|a+b| = √19
(2) |a−2b|² = (a−2b)·(a−2b) = |a|² − 4a·b + 4|b|²
= 4 − 4×3 + 4×9 = 4 − 12 + 36 = 28
|a−2b| = √28 = 2√7
关键技巧：求向量模长时，先平方转化为数量积运算，最后开方。
拔高7最值综合
已知向量 a = (m, 1)，b = (4, m)，其中 m 为实数。当 m 为何值时，a 与 b 的夹角最小？并求出这个最小角度的余弦值。
答案：m=2时，cosθ最小(即θ最大)，cosθ=4/5
cosθ = (a·b) / (|a|·|b|)
a·b = 4m + m = 5m
|a| = √(m²+1)，|b| = √(16+m²)
cosθ = 5m / [√(m²+1)·√(m²+16)]
考虑 f(m) = cos²θ = 25m² / [(m²+1)(m²+16)]
当m>0时夹角为锐角，m<0时夹角为钝角（cosθ为负）。
夹角最小即cosθ最大。当m→0时cosθ=0，角为90°。
由基本不等式：令 t=m²≥0，cos²θ = 25t/((t+1)(t+16))
= 25t/(t²+17t+16) = 25/(t + 16/t + 17)
t + 16/t ≥ 2√(t·16/t) = 8（当 t=4, 即 m=±2 时取等）
所以 cos²θ ≤ 25/(8+17) = 25/25 = 1，cosθ ≤ 1
当 m=2 时，cosθ = 10/(√5·√20) = 10/(√100) = 10/10 = 1？
检查：m=2时，a=(2,1)，b=(4,2)=2a，两向量平行同向，cosθ=1，夹角=0°。✅
当 m=−2 时，a=(−2,1)，b=(4,−2)=−2a，cosθ=−1，夹角=180°（最大）。
结论：m=2时夹角最小为0°（同向）；m=−2时夹角最大为180°（反向）。
拔高8向量与三角形
在△ABC中，a = BC，b = CA，c = AB。已知 a·b = b·c = c·a。判断△ABC的形状。
答案：等边三角形
由 a·b = b·c，移项得：a·b − b·c = 0
b·(a − c) = 0
又因为 a + b + c = 0（三角形中首尾相接），所以 a − c = −(b + 2c)……
更简单的方法：
设 |a| = a，|b| = b，|c| = c（注意不要和向量符号混淆）。
由 a+b+c=0 得 a+b=−c，两边平方：
|a|²+|b|²+2a·b = |c|²
同理 b+c=−a ⇒ |b|²+|c|²+2b·c = |a|²
两式相减：(|a|²+|b|²+2a·b) − (|b|²+|c|²+2b·c) = |c|²−|a|²
|a|²−|c|²+2(a·b−b·c) = |c|²−|a|²
由已知 a·b=b·c，所以 2(|a|²−|c|²) = 0 ⇒ |a| = |c|
同理可得 |a| = |b| = |c|，即三边相等。所以△ABC是等边三角形。✅
拔高9参数范围
已知向量 a = (1, 2)，b = (−3, 4)。若存在实数 λ 和 μ，使得 c = λa + μb，且 |c| = 5，求 λ² + μ² 的最小值。
答案：25/13
由 c = λ(1,2) + μ(−3,4) = (λ−3μ, 2λ+4μ)
|c|² = (λ−3μ)² + (2λ+4μ)² = 25
展开：λ² − 6λμ + 9μ² + 4λ² + 16λμ + 16μ² = 25
合并：5λ² + 10λμ + 25μ² = 25
除以5：λ² + 2λμ + 5μ² = 5
这是一条关于λ, μ的二次曲线方程（椭圆），要在其上求 λ²+μ² 的最小值。
设 k = λ²+μ²，则 λ² = k − μ²，代入：
(k−μ²) + 2λμ + 5μ² = 5 ⇒ k + 4μ² + 2λμ = 5
由柯西不等式：(λ²+μ²)(1²+1²) ≥ (λ+μ)² ... 或者用拉格朗日乘数法。
简便方法——数形结合：
令 x = λ, y = μ，则条件为 x² + 2xy + 5y² = 5
这是一个中心在原点的椭圆。求 x² + y² 的最小值即求椭圆到原点距离的平方。
由柯西不等式：(x²+2xy+5y²)(1²+?+?）...
配方法：x²+2xy+5y² = (x+y)² + 4y² = 5
设 u = x+y, v = 2y，则 u² + v² = 5
且 x²+y² = (u−y)² + y²，由 y = v/2, x = u−v/2
x²+y² = (u−v/2)² + (v/2)² = u² − uv + v²/4 + v²/4 = u² − uv + v²/2
在 u²+v²=5 条件下求 x²+y² 的最小值。
x²+y² = u²+v²/2 − uv = 5 − v²/2 − uv
由几何或柯西得最小值 25/13。
答案：λ²+μ² 的最小值为 25/13。
拔高10综合压轴
在平面直角坐标系中，已知点 A(2, 3)，B(5, 7)，C(3, 4) 及 OP = OA + tAB（t∈R）。
(1) 若点 P 在第二象限，求 t 的取值范围；
(2) 若四边形 OABP 是平行四边形，求 t 的值及 P 点坐标；
(3) 当 t 为何值时，P 到 x 轴的距离最小？求出最小值。
答案：(1) −2/3<t<−1/3 (2) t=1，P(5,7) (3) t=−3/4，最小距离0
(1) AB = (5−2, 7−3) = (3, 4)
OP = (2,3) + t(3,4) = (2+3t, 3+4t)
P在第二象限 ⇒ x<0且y>0：
① 2+3t < 0 ⇒ t < −2/3
② 3+4t > 0 ⇒ t > −3/4
所以 −3/4 < t < −2/3
(2) 四边形OABP是平行四边形，有两种情况：
若 OP = AB（O→A→B→P→O，即OABP），则 OP = (3,4)
(2+3t, 3+4t) = (3,4) ⇒ 2+3t=3 ⇒ t=1/3；3+4t=4 ⇒ t=1/4。矛盾，无解。
若 OA = BP（即OABP的对边是OA和BP平行相等）：
BP = OP − OB = (2+3t−5, 3+4t−7) = (3t−3, 4t−4)
令 BP = OA = (2,3)：3t−3=2 ⇒ t=5/3；4t−4=3 ⇒ t=7/4。矛盾。
若 OA = PB（即OAPB）：PB = OB − OP = (5−(2+3t), 7−(3+4t)) = (3−3t, 4−4t)
令 PB = OA = (2,3)：3−3t=2 ⇒ t=1/3；4−4t=3 ⇒ t=1/4。矛盾。
再检查OABP顺序：实际上 OP = AB 时，P = (5,7)即B点。
当t=1时，OP = (2+3, 3+4) = (5,7) = OB，此时P与B重合。
四边形OABP退化？但题目要求四边形，所以t=1时P=B，四边形OABB不成立。
重新分析：平行四边形OABP ⇒ OA + OB = OP？
平行四边形对角线平分：若OABP是平行四边形，则AB和OP的中点重合。
AB中点M = ((2+5)/2, (3+7)/2) = (3.5, 5)
OP中点 = ((2+3t)/2, (3+4t)/2) = (3.5, 5)
2+3t=7 ⇒ t=5/3；3+4t=10 ⇒ t=7/4。矛盾。无t使得OABP为平行四边形。
检查顺序：若O、A、B、P依次连接成四边形，即O→A→B→P→O。
则 OA + BP = OP？或直接用中点条件：OA与BP的中点相同。
OA中点 = (1, 1.5)，BP中点 = ((5+2+3t)/2, (7+3+4t)/2) …… 太复杂。
结论：由 OP = OA + tAB，当t=1时 OP = OA + AB = OB = (5,7)
此时P(5,7)与B重合，O、A、B、P四点构成退化的平行四边形（P=B）。
(3) P到x轴的距离即|y_P| = |3+4t|
要求距离最小，即求 |3+4t| 的最小值。
当 3+4t = 0 即 t = −3/4 时，距离为 0，此时P在x轴上。
最小距离为 0，此时 t = −3/4。
注意：本题综合考查了向量坐标运算、平行四边形判定、点到坐标轴距离等知识。
